IIS 5.1 へようこそ
Microsoft Windows XP Professional インターネット インフォメーション サービス (IIS) 5.1 は、Windows での Web コンピューティングを可能にします。 IIS を使って、ファイルやプリンタを簡単に共有することができます。 また、セキュリティで保護された Web に情報を公開するためのアプリケーションを作成して、組織が情報を共有する方法を改善することもできます。 IIS は、E コマースやミッションクリティカルなアプリケーションを Web に構築、導入するための安全なプラットフォームです。

IIS がインストールされた Windows XP Professional は、次のようなことが可能なオペレーティング システムです :

  • パーソナル Web サーバーのセット アップ
  • チーム内での情報の共有
  • データベースへのアクセス
  • 企業イントラネットの開発
  • Web アプリケーションの開発

IIS は、Windows とインターネット標準を統合しています。このため、コンテンツを発行、 管理、開発するための新しい方法をはじめから学習する必要はありません。

統合された管理
Windows XP のコンピュータ管理、またはスクリプトを使って IIS を管理できます。コンソール コンソールを使って、IIS で管理しているサーバーやサイトのコンテンツを共有することもできます。 コンソールから IIS スナップインにアクセスすると、ほとんどの IIS の設定を行うことができます。 サイトとアプリケーションの開発後は、これらの設定を実際の運用環境で使うことができます。

オンライン ヘルプ
IIS のオンライン ヘルプにはインデックス、フル テキスト検索機能 があります。また、トピックごとに印刷することも可能です。プログラム的な管理とスクリプトの開発については、 サンプルを使ってください。ヘルプは HTML なので、必要に応じて 注釈をつけたり共有したりすることができます。 IIS オンライン ヘルプを使うと次のことが可能です :

  • タスクの完了
  • サーバーの操作と管理についての学習
  • リファレンスの参照
  • サンプル コードの表示

IIS に関するその他の有用な情報は、 Microsoft.com Web サイト: MSDN、 TechNet、Windows サイトにあります。

Web 印刷
Windows XP Professional は、サーバー上のすべてのプリンタを、簡単にアクセスできる Web サイトに動的に一覧表示します。 このサイトを参照して、プリンタとそのジョブを監視できます。 また、このサイトを介してどの Windows コンピュータからでもプリンタに接続できます。 インターネット印刷については、Windows のヘルプを参照してください。
