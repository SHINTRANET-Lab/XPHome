はじめに

インターネット インフォメーション サービス 5.1 (IIS) は、イントラネットで簡単に情報を公開するための Windows XP の Web サービスです。

[コンテンツ] タブの [同期] ボタン をクリックすると、コンテンツ ペインで表示されているドキュメントに対応する目次が、ナビゲーション ペインに表示されます。[検索] タブを使用してトピックを検索し、トピック リンクをクリックします。[コンテンツ] タブをクリックして [同期] ボタンをクリックします。トピックの場所が目次に表示されます。

  • 「リリース ノート」には、インターネット インフォメーション サービスを適切にインストールして、使用するための最新情報が記載されています。
  • 「IIS のインストール」では、IIS のインストール方法と、コンポーネントの追加や削除方法を説明し、IIS のアンインストール後に残るディレクトリを一覧で示します。
  • 「機能」では、インターネット インフォメーション サービス 5.1 で新たに採用された機能を一覧で示し、旧バージョンから引き継がれる機能を説明します。
  • 「変更箇所」では、IIS のバージョン 4.0、5.0、および 5.1 における重要な変更箇所を一覧で示します。
  • 「Web リソース」では、インターネットで利用できる IIS 関連のリソースを一覧で示します。
  • 「IIS によるサイトのクイック セットアップ」では、既定の設定を使用して、Web サイトと FTP サイトを作成する方法を説明します。
  • 「マニュアルを使用する」では、必要な情報の検索に役立つツールについて説明します。
  • 「IIS スナップイン」では、IIS を管理しオプションを設定する主要ダイアログ ボックスについて説明します。
  • 「よく寄せられる質問」では、インターネット インフォメーション サービスに関する一般的な質問に回答します。
  • 「用語集」では、インターネットとイントラネットの一般的な用語について説明します。

製品サポートについては、「よく寄せられる質問」の「テクニカル サポート」トピックを参照してください。

製品のほかのユーザーと対話できるピアツーピア ニュースグループが利用できます。ニュースグループには、任意のニュース読み取りソフトウェアを使用してアクセスできますが、読み取るにはソフトウェアの構成が必要な場合があります。ニュース サーバーを指定するよう指示されたら、「msnews.microsoft.com」を指定してください。アカウント名またはパスワードを入力する必要はありません。ニュースグループに投稿する前に、参加のルールをお読みください。

IIS 5.0 および 5.1 に関する話題は、microsoft.public.inetserver.iis を使用してください。
