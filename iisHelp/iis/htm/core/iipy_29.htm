[サイト オペレータ] プロパティ シート

このプロパティ シートは、サイトに対するオペレータ権限を持つ Windows のユーザー アカウントを定義するときに使用します。

[Web サイト オペレータ]

オペレータの一覧にユーザー アカウントを追加するには、[追加] をクリックします。

選択したユーザー アカウントを削除するには、[削除] をクリックします。

オペレータは、次のような Web サイトの管理作業を行うことができます。

  • Web サーバーのアクセス権およびログの収集を設定する。
  • 既定の Web ドキュメントおよびフッターを変更する。
  • Web コンテンツの有効期限、HTTP ヘッダー、およびコンテンツの規制機能を有効にする。

オペレータ権限を持つユーザーでも、Administrator 権限を持たない場合は、次のような特別な管理作業を行うことはできません。

  • Web サイトの識別情報を変更する。
  • 匿名ユーザーの名前とパスワードを再構成する。
  • 帯域幅を調整する。
  • 仮想ディレクトリを作成する。または仮想ディレクトリのパスを変更する。
  • アプリケーションの分離を選択する。または選択を解除する。
