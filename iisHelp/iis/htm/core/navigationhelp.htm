IIS のヘルプ機能を使用する

インターネット インフォメーション サービスのオンライン マニュアルには、必要な情報を簡単に検索するための機能がいくつかあります。IIS のヘルプは、マウスまたはキーボードを使用することによってナビゲーションできます。

目次

  • 使用できるすべての機能とトピックが記載された目次を、ナビゲーションのウィンドウに表示します。
  • 最上位レベルのトピック タイトルを選択またはクリックすると、そのセクションの内容を記述するページがトピックのウィンドウに表示されます。
  • トピック タイトルのプラス記号を選択またはクリックすると、目次が開いてそのセクションのすべてのトピックが表示されます。

検索

  • ナビゲーション ウィンドウの [検索] タブで、[検索文字列] テキスト ボックスに単語または句を入力して [検索] ボタンを選択します。
  • [検索の種類] テキスト ボックスから検索の種類を選択します。既定の [標準検索] では、語または句のいくつかのバリエーションを検索します。[完全一致]、[いずれかの語を含む]、[すべての語を含む]、または [論理検索] を選択して、検索の範囲を限定することができます。

索引

  • ナビゲーション ウィンドウの [索引] タブで、テキスト ボックスに語を入力して、その語に最も近いキーワードを表示します。
  • キーワードをダブルクリックするか、キーワードを選択して Enter キーを押すと、そのファイルを表示するか、またはキーワードに関連する情報が含まれたトピックの一覧を表示します。

印刷

必要に応じてトピックを印刷するには、[印刷] を選択するか、クリックします。

索引および検索の結果で目次を同期する

索引ツールまたは検索ツールからトピックを選択した後で、そのトピックが目次内のどこに表示されるかを確認できます。[索引] タブまたは [検索] タブからトピックを表示すると、トピックはマニュアル ウィンドウの右側ウィンドウに表示されます。

  • [索引] タブまたは [検索] タブからトピックを表示した後に、[目次] タブを選択またはクリックして、[同期] ボタンを選択またはクリックします。

キーボードを使用した IIS ヘルプのナビゲーション方法

Tab キーをナビゲーション キーとして使用できます。

  • Tab キーを押すと、ウィンドウからウィンドウに移動でき、目次を含むウィンドウの [目次]、[索引]、および [検索] の各タブを含めて、各ウィンドウ内のリンクを移動できます。
  • Shift + Tab キーを押すと、移動の方向が逆になります。

目次ウィンドウからトピックまたは項目を選択する

  • トピックまたは項目を選択して Enter キーを押すと、表示されます。選択した項目が目次内の親項目の場合は、含まれるトピックが表示されます。
  • 目次内を移動するには、↑ キーまたは ↓ キーを使用するか、PageUp キーおよび PageDown キーを使用します。

索引ウィンドウからトピックを選択する

  • テキスト ボックスに用語を入力するか、表示されている索引から検索する用語を選択します。
  • Alt + D キーを押すと、トピックまたは追加の選択肢が表示されます。

検索ウィンドウからトピックを選択する

  • [検索文字列] テキスト ボックスに検索する用語を入力します。
  • Tab キーを押してリスト ボックスに移動し、Alt + ↑ キーまたは Alt + ↓ キーを押して検索の種類を選択します。
  • Tab キーを使用して [検索] ボタンに移動し、Enter キーを押して検索結果を表示します。
