Windows XP による発行

Windows XP Professional から、別のサーバー上にある WebDAV 発行ディレクトリに接続できるようになりました。Windows XP Professional の既定の設定では、IIS 5.1 はインストールされません。したがって、インストールのオプションで IIS 5.1 を選択しなかった場合は、手動でインストールする必要があります。IIS のインストール方法については、「IIS のインストール」を参照してください。

Windows XP を使用した WebDAV ディレクトリへの接続
  1. [スタート] ボタンをクリックし、[マイ コンピュータ] をクリックして、[マイ ネットワーク] をクリックします。
  2. [ネットワーク プレースの追加] をダブルクリックします。
  3. ネットワーク プレースの追加 ウィザードで、接続する WebDAV ディレクトリの URL を入力します。
  4. [次へ] をクリックします。
  5. 接続するディレクトリの登録名を入力します。
  6. [完了] をクリックします。
  7. [マイ ネットワーク] をダブルクリックします。WebDAV サイトの Web フォルダの登録名が表示されます。たとえば、登録名として、My Publishing Folder と入力すると、"My Publishing Folder へのショートカット" と表示されます。
  8. Web フォルダの名称をダブルクリックします。
  9. [My Publishing Folder へのショートカット] をダブルクリックします。ディレクトリに含まれるファイルの一覧が表示されます。
