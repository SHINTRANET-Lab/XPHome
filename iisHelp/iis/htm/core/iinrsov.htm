名前解決

利用者が目的のサイトにアクセスしてページを表示するには、インターネット上またはローカルのイントラネット上で固有の識別情報が、そのサイトに設定されていなければなりません。サイトの "わかりやすい名前" を解決して、実際にアクセスするためのアドレスを取得する方法は、ネットワークで利用可能な名前解決システムによって決まります。

ここで説明する項目は、次のとおりです。

  • 「名前解決について」では、利用可能な各種名前付けシステム、各システムの動作方法、状況に適したシステムの選択、およびサイトに名前と IP アドレスを割り当てるための規則について説明します。
  • 「コンピュータの識別情報を確認する」では、Windows のコンピュータ名と IP アドレスを確認する方法について説明します。

関連するその他の項目

  • Web サイトに識別情報を割り当てる方法については、「Web サイトに名前を付ける」を参照してください。
