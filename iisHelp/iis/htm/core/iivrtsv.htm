Web サイトと FTP サイト

本来、www.microsoft.com などのドメイン名は、個々のコンピュータを表すものでした。現在では、Windows 2000 Server が稼働している 1 台のコンピュータで、複数の Web サイトや FTP サイトを同時に運用し、複数のコンピュータがあるように見せることができます。各 Web サイトは、1 つまたは複数のドメイン名のホストとなることができます。各サイトが別々のコンピュータのように動作するので、サイトを "仮想サーバー" と呼ぶこともあります。

ここで説明する項目は、次のとおりです。

  • 「Web サイトおよび FTP サイトについて」では、Web サイトおよび FTP サイトと、それらのプロパティの概念を説明します。
  • 「サイトを開始および停止する」では、IIS スナップインによるサイトの開始と停止方法について説明します。
  • 「FTP メッセージとディレクトリ表示スタイルを設定する」では、カスタマイズしたメッセージと、FTP クライアント用のディレクトリ一覧形式の作成方法について説明します。
  • 「Web サイトに名前を付ける」では、Web サイトに一意の組み合わせのホスト ヘッダー名、IP アドレス、およびポートを割り当てる方法について説明します。
  • 「IIS を再起動する」では、IIS スナップインを使用し、インターネット サービスを停止および再起動する方法について説明します。
  • 「継承された既定の設定を変更する」では、サイトのグローバルな既定の設定を変更する方法について説明します。
  • 「メタベースのバックアップ」では、IIS の構成設定をバックアップする方法について説明します。
  • 「メタベースを復元する」では、IIS の構成設定を復元する方法について説明します。

関連するその他の項目

  • ネットワークの名前付けシステムについては、「名前解決」を参照してください。
  • セキュリティ設定の拡張については、「セキュリティ」を参照してください。
  • 仮想ディレクトリの追加については、「仮想ディレクトリを作成する」を参照してください。
