[コンテンツの規制] プロパティ シート

このプロパティ シートは、特定の Web サイト、ディレクトリ、またはファイルに対して、Web サーバーのコンテンツ規制機能を構成するときに使用します。

Web サーバーの Platform for Internet Content Selection (PICS) システムでは、Recreational Software Advisory Council (RSAC) が開発した規制システムを採用しています。このシステムでは、暴力、性的な描写、および不快な言葉のレベルに応じてコンテンツを規制します。

コンテンツの規制を Web コンテンツに適用するには、[このリソースの規制を有効にする] チェック ボックスをオンにします。

コンテンツの規制を設定するには、[分類] ボックスで、[言葉]、[ヌード]、[セックス]、[暴力] の中から、規制するカテゴリをクリックします。そのカテゴリについて内容が不快感を与える可能性のレベルを設定するには、規制のスライダを使用します。各設定で、規制レベルの説明が表示されます。

コンテンツを規制した人を明らかにするには、[電子メール アドレス] ボックスに電子メールのアドレスを入力します。

規制されたコンテンツに有効期限を設定するには、[規制の有効期限] ボックスの右にあるボタンをクリックします。カレンダーが表示されるので、有効期限の日付を指定します。
