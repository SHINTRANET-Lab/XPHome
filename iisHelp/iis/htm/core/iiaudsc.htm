監査

セキュリティ監査技術を使用して、ユーザーの行動を追跡したり、NTFS ディレクトリおよびファイルにアクセスする権限のない試みを検出することができます。監査できる行動は、次のとおりです。

  • 成功と失敗の両方を含む、すべてのログオン要求
  • 制限されたアカウントにアクセスを試みたユーザーの要求
  • 制限されたコマンドの実行を試みたユーザーの要求

ここでは、次の項目について説明します。

  • 「監査について」では、Web サーバーのセキュリティの監視と評価について説明します。
  • 「監査を構成および監視する」では、セキュリティ ログの作成について説明します。
  • 「権限のないアクセスを検出する」では、セキュリティ ログを解析して権限のないアクセスの痕跡を検出する方法について説明します。

関連するその他の項目

  • ユーザーによる Web サイトおよび FTP サイトの利用状況のログを収集する方法については、「サイトの利用状況のログを収集する」を参照してください。
  • Web サイトおよび FTP サイトへのアクセスを制御する方法については、「アクセス制御」を参照してください。
