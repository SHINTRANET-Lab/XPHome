IIS スナップインを開く

Microsoft 管理コンソール (MMC) 内にホストされているインターネット インフォメーション サービス スナップインは、Web サイトまたは FTP サイトのコンテンツおよびアクセスを制御するために、サーバー管理機能を提供します。以前のバージョンでは、インターネット サービス マネージャと呼ばれていました。

インターネット インフォメーション サービス スナップインを開くには


  1. [スタート] メニューの [ファイル名を指定して実行] をクリックします。[ファイル名を指定して実行] ダイアログ ボックスが表示されます。
  2. [名前] ボックスに「inetmgr」と入力して Enter キーを押します。インターネット インフォメーション サービス スナップインが表示されます。
または、[コンピュータの管理] ダイアログ ボックスから IIS にアクセスすることもできます。この方法で IIS にアクセスすると IIS スナップインの管理オプションは使用できませんが、素早くアクセスでき、Web サイト用のいくつかの管理オプションが使用できます。
  1. [マイ コンピュータ] を右クリックして、[管理] をクリックします。[コンピュータの管理] ダイアログ ボックスが表示されます。
  2. 左側のウィンドウで、[サービスとアプリケーション] ツリーをクリックします。
  3. [インターネット インフォメーション サービス] をクリックします。ユーザーの Web サイトの名前と状態が右側のウィンドウに表示されます。

    4. 注    左側ウィンドウのインターネット インフォメーション サービス ツリーおよびその下の Web サイト ツリーをクリックすると、その Web ページのディレクトリと仮想ディレクトリの一覧が表示されます。

IIS スナップインの使い方および管理方法の詳細については、「管理」を参照してください。
