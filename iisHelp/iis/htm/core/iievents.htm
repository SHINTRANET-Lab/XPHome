IIS の Windows イベント

IIS がイベント ビューアに生成するイベントにより、IIS のパフォーマンスを確認することができます。イベント ビューアは、重要度の高いものから順に、エラー イベント、警告イベント、および情報イベントを追跡します。イベント ビューアのログは、すべての Windows サービス、およびログオン時間や接続時間などのプロセスに関する監査記録です。

ここで説明する項目は、次のとおりです。

  • 「Windows イベントについて」では、Windows イベントに関連する項目を説明します。
  • 「Web サービス イベント」では、Web サービスで生成されるイベントの一覧を示します。
  • 「FTP サービス イベント」では、FTP サービスで生成されるイベントの一覧を示します。
  • 「Active Server Pages イベント」では、ASP で生成されるイベントの一覧を示します。

関連するその他の項目

  • クライアントのブラウザで表示されるエラー メッセージについては、「カスタム エラー メッセージ」を参照してください。
  • クライアントの要求に関するログ収集については、「サイトの利用状況のログを収集する」を参照してください。
