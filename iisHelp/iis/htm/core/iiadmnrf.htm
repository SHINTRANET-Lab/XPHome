管理者用リファレンス

ここでは、レジストリ構成プロパティ、メタベース、IIS の Windows イベント、カウンタ、タイムアウト、およびログ収集プロパティの詳細について説明します。このリファレンスは、管理作業用です。

ここでは、次の項目について説明します。

  • 「レジストリ」では、IIS レジストリ キーの構成プロパティについて説明します。
  • 「メタベース」では、メタベースの構成プロパティについて説明します。
  • 「IIS の Windows イベント」は、IIS によって Windows のイベント ログに記録されるイベントの一覧です。
  • 「その他のリファレンス」は、タイムアウトやログ収集プロパティの既定の設定、およびカウンタに関する情報のリファレンスです。

関連するその他の項目

  • 定期的なサーバー管理については、「サーバーの管理」を参照してください。
  • IIS Admin オブジェクトについては、「IIS Admin オブジェクトを使用する」を参照してください。
  • 管理スクリプトについては、「管理スクリプト」を参照してください。
