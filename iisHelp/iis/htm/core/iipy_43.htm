[ODBC のプロパティ]

IIS から ODBC のログを収集するには、事前に次の作業が必要です。

  1. Microsoft Access や Microsoft SQL Server などの ODBC 準拠のデータベース ソフトウェアを使って、管理するコンピュータ上にデータベースを作成します。
  2. 作成したデータベースに、必要なフィールドを含んだテーブルを作成します。次に、データ ソース名を作成します。詳細については、Windows のマニュアルを参照してください。
  3. このダイアログ ボックスを使って、IIS にデータ ソース名とテーブル名を指定します。
  4. データベースへのアクセスに、ユーザー名とパスワードが必要である場合は、それらも指定します。
