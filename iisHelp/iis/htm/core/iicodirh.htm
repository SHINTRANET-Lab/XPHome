ホーム ディレクトリを変更する

各 Web サイトには、ホームディレクトリが 1 つ必要です。同じコンピュータに Web サイトと FTP サイトの両方をセットアップする場合は、WWW と FTP の各サービスに、それぞれ異なるホーム ディレクトリを指定する必要があります。WWW サービスの既定のホーム ディレクトリは、C:\Inetpub\Wwwroot です。FTP サービスの既定のホーム ディレクトリは、C:\Inetpub\Ftproot です。ほかのディレクトリをホーム ディレクトリに指定することもできます。

ホーム ディレクトリを変更するには
  1. IIS スナップインで、Web サイトまたは FTP サイトを選択し、そのプロパティ シートを開きます。
  2. [ホーム ディレクトリ] タブをクリックし、ホーム ディレクトリの場所を指定します。次のいずれかをホーム ディレクトリとして選択できます。
    • そのコンピュータにあるディレクトリ
    • 別のコンピュータにある共有ディレクトリ
    • URL へのリダイレクト

      • 注   この URL を要求したブラウザは、新しい URL に転送されます。FTP のディレクトリはリダイレクトできません。

  3. テキスト ボックスに、ホーム ディレクトリのパス名、共有名、または URL を入力します。

    4. 注   ネットワーク共有のディレクトリを指定する場合、リソースにアクセスするためのユーザー名とパスワードの入力が必要な場合があります。IUSR_computername が推奨されるアカウントです。サーバーの管理者アクセス権が与えられているアカウントを使うと、クライアントからサーバーを操作できるようになります。このことは、ネットワークのセキュリティ上たいへん危険です。
