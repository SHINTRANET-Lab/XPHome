WebDAV の発行

ここでは、IIS サーバー上での WebDAV 発行ディレクトリの設定の方法と、クライアントがサーバーに接続し、ファイルを編集、操作する方法について説明します。

ここで説明する項目は、次のとおりです。

  • 「WebDAV について」では、リモートでユーザーがファイルを発行し、操作するディレクトリを WebDAV により作成する方法について説明します。
  • 「発行ディレクトリを作成する」では、WebDAV 発行ディレクトリの設定の方法について説明します。
  • 「WebDAV セキュリティの管理」では、WebDAV サイトのセキュリティ保護に、IIS と Windows XP のセキュリティを有効に利用する方法を説明します
  • 「WebDAV ディレクトリを検索する」では、ユーザーが、コンテンツとドキュメント プロパティの検索を行えるように、WebDAV ディレクトリを構成する方法について説明します。
  • 「ファイルを発行し管理する」では、Windows XP、Internet Explorer、および Office 2000 により、コンテンツの発行とファイルの操作を行う方法について説明します。

注   WebDAV は、HTTP 1.1 用に設計されたものです。したがって、FTP サイトなどの非 HTTP サービスでは利用できません。また、WebDAV は、現在 IIS 5.0 および 5.1 のみに実装されています。
