IIS によるサイトのクイック セットアップ

Windows XP Professional をインストールすると、IIS は既定の Web サイトおよび FTP サイトを作成します。ここでは、既定のサイトに情報を公開する方法について説明します。

Web サイトでコンテンツを公開するには
  1. Web サイトのホーム ページを作成します。Web サイトの作成に利用できるツールの詳細については、「オーサリング ツールの選択」を参照してください。
  2. ホーム ページ ファイルに、Default.htm または Default.asp という名前を付けます。
  3. IIS の既定の Web 発行ディレクトリにホーム ページ ファイルをコピーします。既定の Web 発行ディレクトリは、ホーム ディレクトリとも呼ばれます。セットアップで用意されているホーム ディレクトリは、\Inetpub\Wwwroot です。
  4. ネットワークで名前解決システム (通常は DNS) が使われている場合、利用者はブラウザのアドレス バーにコンピュータ名を入力するだけで、サイトにアクセスできます。名前解決システムが使われていない場合、利用者はコンピュータの IP アドレスを入力する必要があります。詳細については、「名前解決について」を参照してください。
FTP サイトにコンテンツを公開するには
  1. 既定の FTP 発行ディレクトリに、ファイルをコピーまたは移動します。セットアップで用意されている既定のディレクトリは、\Inetpub\Ftproot です。
  2. ネットワークで名前解決システム (通常は DNS) が使われている場合、利用者はブラウザのアドレス バーに、「ftp://」に続けてコンピュータ名を入力し、サイトにアクセスすることができます。名前解決システムが使われていない場合、利用者は ftp:// およびコンピュータの IP アドレスを入力する必要があります。FTP サイトの外観をカスタマイズするには、「FTP メッセージとディレクトリ表示スタイルを設定する」を参照してください。
