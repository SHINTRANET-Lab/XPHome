アクセス制御

Web サーバーとそのリソースへのアクセスを許可する、ユーザーおよびコンピュータを制御することができます。NTFS および Web サーバーのセキュリティ機能を使用して、ディレクトリとファイルに対する特定のアクセス権をユーザーに割り当てることができます。

ここでは、次の項目について説明します。

  • 「アクセス制御について」では、ファイルとディレクトリへのアクセスの制限について、概要を説明します。
  • 「NTFS でファイルを保護する」では、サーバーで NTFS のセキュリティ機能を使用するように構成する手順について説明します。
  • 「ディレクトリおよびファイルに対する NTFS のアクセス権を設定する」では、Windows ユーザー アカウントに割り当てるファイルやディレクトリへのアクセス権のレベル設定が可能な NTFS の構成方法について説明します。
  • 「Web サーバーのアクセス権を設定する」では、すべてのユーザーに適用される Web および FTP のコンテンツに対する制限を設定する方法について説明します。
  • 「データベースへのアクセスを制御する」では、権限のないアクセスからデータベース ファイルを保護する方法について説明します。

関連するその他の項目
