Web ページにフッターを追加する

Web サーバーが送信するすべての Web ページの下部に HTML 形式のファイルを自動的に挿入するよう、Web サーバーを構成することができます (ASP ページに対するフッターの追加は、サポートされません)。たとえば、Web ページに簡単なメッセージ文やロゴ画像を追加するための HTML 書式命令を入れることができます。詳細については、「Web サイト管理について」および「サーバー側インクルード」を参照してください。

注   ドキュメント フッターを使用すると、Web サーバーのパフォーマンスが低下する場合があります。 特に、頻繁にアクセスされる Web ページの場合は、その可能性があります。

Web ページにフッターを追加するには
  1. HTML でフッター ファイルを作成し、Web サーバーのハード ディスクに保存します。

    2. 注   フッター ファイルは、完全な HTML ドキュメントである必要はありません。フッターの内容を表示する書式に必要な HTML だけを入れるようにします。たとえば、各 Web ページの下部に組織名を追加するフッター ファイルには、テキスト、およびそのテキストのフォントと色を表示するために必要な HTML タグを入れるようにします。

  2. IIS スナップインで、Web サイト、ディレクトリ、またはファイルを選択し、そのプロパティ シートを開きます。
  3. [ドキュメント] タブをクリックし、[ドキュメント フッターを有効にする] チェック ボックスをオンにします。
  4. テキスト ボックスに、フッター ファイルの完全なローカル パスを入力するか、[参照] をクリックしてファイルを選択します。
