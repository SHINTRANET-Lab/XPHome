仮想ディレクトリを作成する

Web サイトに含まれるファイルが、ホーム ディレクトリとは異なるディレクトリ、またはほかのコンピュータにある場合、仮想ディレクトリを作成してこれらのファイルを Web サイトに取り込む必要があります。ほかのコンピュータのディレクトリを使用するには、ディレクトリの UNC (Universal Naming Convention) 名を指定し、アクセス権を得るのに使用するユーザー名とパスワードを入力する必要があります。

仮想ディレクトリを作成するには
  1. IIS スナップインで、ディレクトリを追加する Web サイトまたは FTP サイトを選択します。
  2. [操作] メニューの [新規] をポイントし、[仮想ディレクトリ] をクリックします。
  3. 仮想ディレクトリの作成ウィザードに従って操作し、仮想ディレクトリを作成します。

注   NTFS を使用している場合は、Windows エクスプローラでディレクトリを右クリックして、[共有] をクリックし、[Web 共有] プロパティ シートをクリックして、仮想ディレクトリを作成することもできます。

仮想ディレクトリを削除するには
  1. IIS スナップインで、削除する仮想ディレクトリを選択します。
  2. [操作] をクリックし、[削除] をクリックします。仮想ディレクトリを削除しても、対応する物理ディレクトリまたはファイルは削除されません。
