コンピュータの識別情報を確認する

[Web サイト] プロパティ シートまたは [FTP サイト] プロパティ シートの Web サイトまたは FTP サイトの説明の中に、表示されるはずの名前またはアドレスがない場合は、次の方法でコンピュータのネットワーク名とアドレスを確認できます。

  • IP アドレスを確認するには、コマンド プロンプトで「ipconfig」と入力し、Enter キーを押します。コンピュータの現在の IP アドレス、サブネット マスク、およびデフォルト ゲートウェイが表示されます。
  • コンピュータ名を確認するには、[マイ コンピュータ] のプロパティ シートを開きます。[ネットワーク ID] タブにコンピュータ名が表示されます。
