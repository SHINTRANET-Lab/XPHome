[拡張プロパティ]

ログに記録するフィールド (項目) を選択することで、W3C 拡張形式のログ収集をカスタマイズすることができます。ログのサイズを小さく抑えるには、不要なフィールドを除外します。

選択できるフィールドは、次のとおりです。

  • [日付] アクティビティが発生した日付です。
  • [時間] アクティビティが発生した時刻です。

[拡張プロパティ]

  • [クライアント IP アドレス] サーバーにアクセスしたクライアントの IP アドレスです。
  • [ユーザー名] サーバーにアクセスしたユーザーの名前です。
  • [サービス名] クライアント コンピュータで実行されていたインターネット サービスです。
  • [サーバー名] ログ エントリが生成されたサーバーの名前です。
  • [サーバー IP アドレス] ログ エントリが生成されたサーバーの IP アドレスです。
  • [サーバー ポート] クライアントの接続先のポート番号です。
  • [メソッド] クライアントが実行しようとした GET コマンドなどの処理です。
  • [URI Stem] HTML ページ、CGI プログラム、スクリプトなどの、アクセスされたリソースです。
  • [URI クエリ] クライアントが実行を試みたクエリです。これは検索に使われた 1 つまたは複数の文字列です。
  • [プロトコルの状態] HTTP 関連の処理の状態です。
  • [Win32 の状態] Windows 関連の処理の状態です。
  • [送信バイト数] サーバーが送信したバイト数です。
  • [受信バイト数] サーバーが受信したバイト数です。
  • [所要時間] 処理にかかった時間です。
  • [プロトコル バージョン] クライアントが使用するプロトコル (HTTP、FTP) のバージョンです。HTTP の場合は、HTTP 1.0 または HTTP 1.1 になります。
  • [ユーザー エージェント] クライアントが使用したブラウザです。
  • [Cookie] 送受信された cookie の内容です。
  • [履歴] ユーザーがサイトを訪れる前に表示していたリンク元のサイトです。

[プロセス総計]

  • [プロセス イベント] イベントを発生させたプロセスの種類で、CGI またはアウトプロセス アプリケーションのいずれかです。記録される種類は、CGI、Application、または All です。
  • [プロセスの種類] 発生したイベントです。イベントの種類は、次のとおりです。Site-Stop、Site-Start、Site-Pause、Periodic-Log、Interval-Start、Interval-End、Interval-Change、Update、Eventlog-Limit、Priority-Limit、Process-Stop-Limit、Site-Pause-Limit、Eventlog-Limit-Reset、Priority-Limit-Reset、Process-Stop-Limit-Reset、Site-Pause-Limit。
    • Site-Stop Web サイトが停止されました。
    • Site-Start Web サイトが開始または再開されました。
    • Site-Pause Web サイトが一時停止されました。
    • Periodic-Log 管理者が指定した間隔で定期的に記録されるログです。
    • Reset Interval-Start リセット間隔が開始しました。
    • Reset Interval-End リセット間隔の終わりに達し、リセットされました。
    • Reset Interval-Change Web サイトの管理者が、リセット間隔の値を変更しました。
    • Update 次のいずれかのイベントが発生しました。ログの間隔の変更。Interval イベントの発生、あるいはサイトの停止、開始、または一時停止。
    • Eventlog-Limit CGI およびアウト プロセス アプリケーションに対する Web サイトの CPU 使用率が、管理者が設定したイベント ログの制限に達したため、Web サイトのイベント ログが作成されました。
    • Priority-Limit 管理者が設定した低優先度の制限に達したため、Web サイトの CGI またはアウト プロセス アプリケーションの優先度が "低" に設定されました。
    • Process-Stop-Limit 管理者が設定したプロセス停止の制限に達したため、Web サイトの CGI またはアウト プロセス アプリケーションが停止されました。
    • Site-Pause-Limit 管理者が設定したサイト一時停止の制限に達したため、CGI またはアウト プロセス アプリケーションを持つ Web サイトが一時停止されました。
    • Eventlog-Limit-Reset リセット間隔が終了、またはイベント ログの制限が、手動で変更されました。
    • Priority-Limit-Reset リセット間隔が終了、または優先度の制限が、手動で変更されました。
    • Process-Stop-Limit-Reset リセット間隔が終了、またはプロセス停止の制限が、手動で変更されました。
    • Site-Pause-Limit リセット間隔が終了、またはサイトの一時停止の制限が、手動で変更されました。
  • [ユーザーの総使用時間] サイトが現在の間隔の中で使用した、ユーザー モードのプロセッサ時間の累計です。単位は秒です。
  • [カーネルの総使用時間] サイトが現在の間隔の中で使用した、カーネル モードのプロセッサ時間の累計です。単位は秒です。
  • [ページ違反の総数] メモリ ページ フォルトを起こしたメモリ参照の総数です。
  • [プロセスの総数] 現在の間隔の中で作成された、CGI またはアウト プロセス アプリケーションの総数です。
  • [有効なプロセス] ログが記録されたときに実行されていた、CGI およびアウト プロセス アプリケーションの総数です。
  • [切断されたプロセスの総数] 現在の間隔の中で、プロセス調整のために停止された CGI およびアウト プロセス アプリケーションの総数です。
