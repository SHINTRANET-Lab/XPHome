IIS へのアクセス方法

ここで説明するショートカット キー操作やキーボード コマンドは、IIS 固有のものです。Windows のショートカット キーの一覧は、Windows ヘルプのキーワードで "Windows 2000 のショートカット キー" の項目を検索し、説明を参照してください。Microsoft 管理コンソール (MMC) の操作方法の詳細については、Windows ヘルプのキーワードで "MMC" の項目を検索し、説明を参照してください。

IIS スナップインにアクセスするには
  1. [スタート] をクリックし、次に [ファイル名を指定して実行] をクリックします。[ファイル名を指定して実行] ダイアログ ボックスが表示されます。
  2. 「mmc %systemroot%\system32\inetsrv\iis.msc」と入力します。
  3. Enter キーを押します。インターネット インフォメーション サービス スナップインが表示されます。

注   IIS スナップインが表示されない場合、コマンドの入力が適切であるかどうかを確認します。mmc の後 (% 記号の前) にはスペースが 1 つ必要です。

[コンピュータの管理] ウィンドウからも IIS にアクセスすることができます。この方法で IIS にアクセスすると、IIS スナップインに備わっている各種の管理機能が使用できません。迅速にアクセスできますが、Web サイトに関する管理機能が制限されます。

  1. [スタート] をクリックし、次に [マイ コンピュータ] を右クリックします。ドロップダウン リストから [管理] をクリックします。[コンピュータの管理] ウィンドウが表示されます。
  2. 左側のウィンドウで、[サービスとアプリケーション] ツリーを展開します。
  3. [インターネット インフォメーション サービス] をクリックします。ユーザーの Web サイトの名前と状態が右側のウィンドウに表示されます。

注  目的の Web ページのディレクトリと仮想ディレクトリの一覧が表示されるまで、左側のウィンドウで [インターネット インフォメーション サービス] ツリーとそれ以降の Web サイトのツリーを展開します。

特定のプロパティ シートにアクセスするには
  1. IIS スナップインで、方向キーを使用して、管理するサイト、仮想ディレクトリ、またはファイルを選択します。ファイルにアクセスするには、詳細ウィンドウに切り替える必要があります。IIS スナップインでコンソール ツリーと詳細ウィンドウを切り替えるには、Alt キーを使用します。
  2. 選択した項目のショートカット メニューを表示するには、Shift キーを押しながら F10 キーを押します。

    3. 注   [表示] メニューを表示する場合は、Ctrl キーを押しながら A キーを押します。

  3. ↑ キーまたは ↓ キーを使用して [プロパティ] を選択します。Enter キーを押します。

    4. 注   「P」と入力すると、サイトは一時停止します。

  4. プロパティ シートのタイトルが強調表示されるまで、Tab キーを押します。
  5. → キーまたは ← キーを使用して、同じ行にあるプロパティ シートを選択します。
  6. 別の行のプロパティ シートに移動するには、↑ キーまたは ↓ キーを使用します。
  7. プロパティ シート内で移動するには、Tab キーを押します。項目を選択するには Tab キーを使用するか、Alt キーを押しながら下線の付いた文字のキーを押します。

    8. 注   チェック ボックスをオンまたはオフにする場合は、チェック ボックスを選択してから Space キーを押します。

IIS のオンライン マニュアル内で移動するには
  1. [目次] タブが手前にない場合は、[目次] タブが強調表示されるまで Tab キーを押すか、または Shift キーを押しながら Tab キーを押します。
  2. Enter キーを押してタブを手前に移動します。

    3. 注   この手順を使用して、[目次]、[索引]、および [検索] タブを切り替えることができます。

  3. 目次の項目が強調表示されるまで Tab キーを押します。
  4. ↑ キーまたは ↓ キーを使用して項目を選択します。
    5. .
  5. ブックを開くには、→ キーを使用します。
  6. ブックを閉じるには、← キーを使用します。
  7. ドキュメントを開くには、開くドキュメントを選択し Enter キーを押します。
