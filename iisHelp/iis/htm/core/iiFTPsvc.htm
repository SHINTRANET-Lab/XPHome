FTP サービス

特に指定しない限り、IIS 5.1 をインストールしても FTP サービスはインストールされません。FTP サービスがインストールされると、x:\Inetpub\Ftproot に既定の FTP サイトが作成されます (x は IIS をインストールしたドライブ名です)。IIS をインストールしている状態で FTP サービスをアンインストールした場合、次の手順に従って FTP サービスを再インストールすることができます。

FTP サービスをインストールするには
  1. [コントロール パネル] で [プログラムの追加と削除] をクリックします。
  2. [Windows コンポーネントの追加と削除] をクリックします。
  3. 一覧から [インターネット インフォメーション サービス (IIS)] を選択して、[詳細] をクリックします。
  4. [FTP (File Transfer Protocol) サービス] チェック ボックスをオンにします。
  5. [OK] をクリックします (Windows XP の CD を使用するか、インストールするネットワーク上のパス名を指定するように要求するメッセージが表示されます)。
  6. [次へ] をクリックします。
  7. [完了] をクリックします。

関連項目

  • IIS をインストールするには、「IIS のインストール」を参照してください。
  • 既定の設定を使用してサイトをすばやく構築するには、「IIS によるサイトのクイック セットアップ」を参照してください。
  • 既存の Web サイトまたは FTP サイトのプロパティを変更するには、「継承された既定の設定を変更する」を参照してください。
  • 仮想ディレクトリを追加するには、「仮想ディレクトリを作成する」を参照してください。
