IIS のインストール

インターネット インフォメーション サービスは、特に指定しない限り Windows XP Professional にはインストールされません。ただし、IIS 5.1 は、既定では、Windows のアップグレード前のバージョンに PWS がインストールされていた場合のみインストールされています。[コントロール パネル] の [プログラムの追加と削除] で、IIS の削除や追加コンポーネントの選択ができます。IIS の使用開始方法については、「IIS によるサイトのクイック セットアップ」および「IIS スナップイン」を参照してください。

IIS のインストールおよびコンポーネントの追加/削除を実行するには

IIS オプション コンポーネントの詳細については、「IIS オプション コンポーネントのインストール」を参照してください。

  1. [スタート] をクリックし、[コントロール パネル] をクリックして、[プログラムの追加と削除] をダブルクリックします。[プログラムの追加と削除] が表示されます。
  2. [プログラムの追加と削除] ダイアログ ボックスの左側の列で、[Windows コンポーネントの追加と削除] をクリックします。
  3. Windows コンポーネント ウィザードで [次へ] をクリックします。
  4. Windows コンポーネントのリストで [インターネット インフォメーション サービス (IIS)] を選択します。
  5. [次へ] をクリックし、その後の指示に従います。

アンインストール後に残るディレクトリ

次のユーザー コンテンツを含むディレクトリは、IIS をアンインストールした後も、コンピュータ上に残ります。

  • \Inetpub
  • \systemroot\Help\IisHelp
  • \systemroot\System32\Inetsrv
