[セキュリティ アカウント] プロパティ シート

このプロパティ シートは、サーバーを使用できるユーザーの制御、および匿名クライアントのログオン要求に使われるアカウントの指定に使用します。

[匿名接続]

クライアントが、"anonymous" というユーザー名を使って FTP サーバーにログオンできるようにするには、このチェック ボックスをオンにします。

匿名の接続に使用される Windows ユーザー アカウントを追加するには、[追加] をクリックします。

Windows ユーザー アカウントを削除するには、アカウントを選択して、[削除] をクリックします。
