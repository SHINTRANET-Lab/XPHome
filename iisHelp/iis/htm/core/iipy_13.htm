[エラー メッセージ] プロパティ シート

このプロパティ シートは、Web サーバー エラーが発生した場合に、クライアントに送信される HTTP エラーをカスタマイズするために使用します。管理者は、標準の HTTP 1.1 エラーを使用できます。これは IIS が提供する詳細なカスタム エラー ファイルです。また、管理者は独自にカスタム エラー ファイルを作成することもできます。

カスタム エラー メッセージのプロパティを変更するには、HTTP エラーを選択し、[種類] 列の、[既定]、[ファイル]、または [URL] をクリックします。

[既定] を選択した場合は、標準の HTTP 1.1 エラーがクライアントに返されます。既定の HTTP 1.1 エラーのテキストが [コンテンツ] 列に表示されます。

[ファイル] を選択した場合は、カスタム エラー ファイルのローカル パスを入力します。

[URL] を選択した場合は、/virdir1/errors/404.htm などの仮想ディレクトリ名で始まる URL へのパスを入力します。この URL は、ローカル サーバーになければなりません。
