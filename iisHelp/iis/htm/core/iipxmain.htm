インターネット サービス マネージャ (HTML)

HTML バージョンのインターネット サービス マネージャは、ブラウザのウィンドウを分割したかたちで表示されます。ブラウザのナビゲーションのウィンドウにはメニュー ビューが表示され、コンテンツのウィンドウにはツリー ビューが表示されます。

ツリー ビューには、管理できるすべてのコンピュータ、Web サイト、FTP サイト、仮想ディレクトリ、およびディレクトリが表示されます。ツリー ビューで項目を選択すると、その項目が強調表示されます。項目を選択してメニュー ビューでコマンドを選択すると、強調表示された項目にそのコマンドが適用されます。

メニュー ビューには、次のコマンドが表示されます。

[新規作成]
Web サイトまたは FTP サイトを新規に作成します。

[削除]
Web サイト、FTP サイト、仮想ディレクトリ、またはディレクトリを削除します。

[名前の変更]
Web サイト、FTP サイト、または仮想ディレクトリの名前を変更します。

[プロパティ]
ツリー ビューで選択した項目のプロパティを表示します。

[開始]
選択した Web サイトまたは FTP サイトを開始します。

[停止]
選択した Web サイトまたは FTP サイトを停止します。

[一時停止]
選択した Web サイトまたは FTP サイトを一時停止します。

[再開]
選択した Web サイトまたは FTP サイトの一時停止を解除します。

[アクセス許可ウィザード]
Web と FTP のアクセス権、その他のアクセス権、および認証方法を設定します。

[マスタ プロパティ]
既存または新規作成するすべての Web サイトおよび FTP サイトが使用する既定値を設定します。

[構成のバックアップ]
インターネット サービス マネージャ (HTML) のセッション中に構成したすべての設定を保存します。

注意   セキュリティが確保されていない場合、インターネット サービス マネージャ (HTML) を安全に利用するには、Secure Sockets Layer (SSL) を使って接続する必要があります。
