[この Web サイトの複数の ID]

このダイアログ ボックスは、Web サイトのそのほかの識別情報を構成するために使用します。

[追加] または [削除] をクリックすると、一覧表示された識別情報を追加または削除できます。既存の識別情報を編集するには、IP アドレスを選択して、[IP アドレス]、[TCP ポート]、[SSL ポート]、または [ホスト ヘッダー名] のうち、必要なフィールドを編集します。

各 Web サイトには、一意の識別特性の組み合わせを設定する必要があります。このため、複数の Web サイトでは、3 つの識別特性 (ドメインまたはホスト ヘッダー名、IP アドレス、およびポート) のうち 2 つまでを共有できますが、1 つは異なるものでなければなりません。また、SSL の証明書にはドメイン名が含まれるので、証明書を使用する Web サイトでは、ほかの Web サイトと IP アドレスを共有できません。

[SSL ポート]

Secure Sockets Layer (SSL) 接続を確立するときに使用するポートを指定します。

Web サーバーの既定の Web サイトでは、このポートは 443 に設定されています。新しい Web サイトを作成する場合は、このポートを明示的に設定する必要があります。SSL ポートの値は、一意で任意の TCP ポート番号に変更できますが、あらかじめユーザーにそのポート番号を知らせておく必要があります。そうしないと、ユーザーはサーバーに接続できなくなります。このオプションを利用できるのは、サーバー証明書がインストールされている場合だけです。
