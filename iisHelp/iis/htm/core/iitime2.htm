IIS スナップインでのタイムアウト設定

このページは、「タイムアウト リファレンス」の補足です。

AspScriptTimeout 値を変更するには
  1. IIS スナップインで、Web サイトまたはアプリケーションの開始点ディレクトリを選択します。
  2. ディレクトリのプロパティ シートを開き、[ホーム ディレクトリ] タブ、[仮想ディレクトリ] タブ、または [ディレクトリ] タブをクリックします。
  3. [構成] をクリックし、次に [アプリケーションのオプション] タブをクリックします。
  4. [ASP のスクリプトのタイムアウト] ボックスに値を入力します。

または
IIS スナップイン | [プロパティ] | [構成] | [アプリケーションのオプション] | [ASP のスクリプトのタイムアウト]
AspSessionTimeout 値を変更するには
  1. IIS スナップインで、Web サイトまたはアプリケーションの開始点ディレクトリを選択します。
  2. ディレクトリのプロパティ シートを開き、[ホーム ディレクトリ] タブ、[仮想ディレクトリ] タブ、または [ディレクトリ] タブをクリックします。
  3. [構成] をクリックし、次に [アプリケーションのオプション] タブをクリックします。
  4. [セッションのタイムアウト] ボックスに値を入力します。
ConnectionTimeout 値を変更するには
  1. IIS スナップインで、Web サイトまたはアプリケーションの開始点ディレクトリを選択します。
  2. ディレクトリのプロパティ シートを開き、[Web サイト] タブをクリックします。
  3. [接続のタイムアウト] ボックスに値を入力します。
CGITimeout 値を変更するには
  1. IIS スナップインで、Web サイトまたはアプリケーションの開始点ディレクトリを選択します。
  2. ディレクトリのプロパティ シートを開き、[ホーム ディレクトリ] タブ、[仮想ディレクトリ] タブ、または [ディレクトリ] タブをクリックします。
  3. [構成] をクリックし、[処理オプション] タブをクリックします。
  4. [CGI スクリプトのタイムアウト] ボックスに値を入力します。
Web サイトのプロセッサ使用率をログ収集するには
  1. IIS スナップインで、プロセッサ使用率をログ収集する Web サイトを選択します。
  2. Web サイトのプロパティ シートを開き、[パフォーマンス] タブをクリックします。
  3. [プロセス調整を有効にする] を選択し、このサイトに対する CPU 時間の上限をパーセントで入力します。
  4. CpuLimitPriority と CpuLimitProcStop を有効にする場合は、[制限を強制する] をクリックします。
