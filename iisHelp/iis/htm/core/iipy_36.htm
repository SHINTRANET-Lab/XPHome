[ドキュメント] プロパティ シート

このプロパティ シートは、サイトの既定の Web ページを定義したり、サイトのドキュメントにフッターを追加するときに使用します。

[既定のドキュメントを有効にする]

ブラウザからの要求にドキュメント名が指定されていない場合に、既定のドキュメントを返すには、このチェック ボックスをオンにします。既定のドキュメントとして指定できるのは、ディレクトリのホーム ページ、またはサイトのドキュメント ディレクトリの一覧を含むインデックス ページです。

既定のドキュメントを新しく追加するには、テキスト ボックスにドキュメントの名前を入力します。既定のドキュメントは、複数指定できます。既定のドキュメントは、名前の一覧に表示されている順に返されます。サーバーは、最初に検出したドキュメントを返します。既定のドキュメントを複数指定する場合は、各ドキュメント名をカンマで区切って入力します。

[ドキュメント フッターを有効にする]

このチェック ボックスをオンにすると、Web サーバーが送信するすべてのドキュメントに HTML 形式のフッターが自動的に追加されます。

フッター ファイルは、完全な HTML ドキュメントでなくてもかまいません。フッターの内容の外観と機能を設定するのに必要な HTML タグが含まれていれば、フッター ファイルとして指定できます。

フッター ファイルのフル パスとファイル名を指定するには、[参照] をクリックします。
