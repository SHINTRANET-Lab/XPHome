継承された既定の設定を変更する

IIS のインストール時に、プロパティ シートのさまざまなプロパティに既定値が割り当てられます。プロパティはサイト レベル、ディレクトリ レベル、またはファイル レベルで設定できます。上位レベル (サイト レベルなど) の設定は、下位レベル (ディレクトリ レベルなど) に自動的に適用、つまり継承されます。ただし、下位レベルで個別にプロパティを変更することも可能です。個々のサイト、ディレクトリ、またはファイルのプロパティが既に変更されている場合は、マスタの既定値を変更しても、個々の設定が自動的に既定の設定に置き換わることはありません。代わりに、サイト、ディレクトリ、またはファイルの設定を、新しい既定に合う設定に変更するかどうかを確認するメッセージが表示されます。

コンピュータに割り当てられるすべてのサイトのマスタ プロパティを設定するには
  1. IIS スナップインで、[コンピュータ] アイコンを選択し、[プロパティ] をクリックして、プロパティ シートを表示します。
  2. [マスタ プロパティ] ボックスで、[WWW サービス] または [FTP サービス] を選択します。
  3. [編集] をクリックします。
  4. 詳細については、各プロパティ シートで [ヘルプ] をクリックし、表示される説明を参照してください。

 

サイトのプロパティを個別に設定するには
  1. IIS スナップインで、Web サイトを右クリックし、[プロパティ] をクリックして、プロパティ シートを表示します。
  2. 詳細については、各プロパティ シートで [ヘルプ] をクリックし、表示される説明を参照してください。
