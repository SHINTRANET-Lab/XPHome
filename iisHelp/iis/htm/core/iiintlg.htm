ログ収集プロパティ リファレンス

このトピックでは、ログ収集プロパティの定義と、W3C 拡張ログ収集形式でのプロパティの値の一覧を表示します。ここでは、次の項目について説明します。

  • W3C 拡張ログ収集の定義 W3C 拡張ログ ファイル形式で使用できる一般的なプロパティ
  • プロセス総計ログ収集の定義 W3C 拡張ログ ファイルに記述されるプロセス総計用の追加プロパティ
  • Microsoft IIログ定義  Microsoft IIS ログ ファイル形式で使用できる一般的なプロパティ
  • NCSA 共通ログ ファイル形式  NCSA 共通ログ ファイル形式で使用できる一般的なプロパティ

W3C 拡張ログ形式の詳細については、World Wide Web Consortium の Web サイトを参照してください。

W3C 拡張ログ収集の定義

プレフィックス 意味
s- サーバーの処理
c- クライアントの処理
cs- クライアントのサーバーに対する処理
sc- サーバーのクライアントに対する処理


フィールド ログ ファイル中のフィールド名 説明
日付 date アクティビティが発生した日付です。
時間 time アクティビティが発生した時刻です。
クライアント IP アドレス c-ip サーバーにアクセスしたクライアントの IP アドレスです。
ユーザー名 cs-username サーバーにアクセスした、認証されているユーザーの名前です。ハイフン (-) で表される匿名ユーザーは含まれません。
サービス名 s-sitename クライアントによってアクセスされたインターネット サービスとインスタンス番号です。
サーバー名 s-computername ログ エントリが生成されたサーバーの名前です。
サーバー IP アドレス s-ip ログ エントリが生成されたサーバーの IP アドレスです。
サーバー ポート s-port クライアントが接続したポート番号です。
メソッド cs-method GET メソッドなどの、クライアントが実行しようとした処理です。
URI Stem cs-uri-stem Default.htm などの、アクセスされたリソースです。
URI クエリ cs-uri-query クライアントが実行しようとしたクエリです。
プロトコルの状態 sc-status HTTP または FTP 関連の処理の状態です。
Win32 の状態 sc-win32-status Windows 関連の処理の状態です。
送信バイト数 sc-bytes サーバーが送信したバイト数です。
受信バイト数 cs-bytes サーバーが受信したバイト数です。
所要時間 time-taken 処理にかかった時間です。
プロトコル バージョン cs-version クライアントが使用したプロトコルの (HTTP、FTP) バージョンです。HTTP では、HTTP 1.0 または HTTP 1.1 のいずれかです。
ホスト cs-host ホスト ヘッダーの内容を表示します。
ユーザー エージェント cs(User-Agent) クライアントが使用したブラウザです。
Cookie cs(Cookie) 送受信された cookie の内容です。
履歴 cs(Referer) ユーザーが利用した直前のサイトです。このサイトが、現在のサイトへのリンクを提供しました。

プロセス総計ログ収集の定義

フィールド ログ ファイル中のフィールド名 説明
プロセス イベント s-event 発生したイベントです。Site-Stop、Site-Start、Site-Pause、Periodic-Log、Interval-Start、Interval-End、Interval-Change、Log-Change-Int/Start/Stop、Eventlog-Limit、Priority-Limit、Process-Stop-Limit、Site-Pause-Limit、Eventlog-Limit-Reset、Priority-Limit-Reset、Process-Stop-Limit-Reset、Site-Pause-Limit-Reset のいずれかです。これらの値の説明については、ここをクリックしてください。
プロセスの種類 s-process-type イベントを発生させたプロセスの種類で、CGI またはアウトプロセス アプリケーションです。記録される種類は、CGI、Application、または All です。
ユーザーの総使用時間 s-user-time 現在の間隔内にサイトが使用した User Mode プロセッサ時間の累計 (%) です。
カーネルの総使用時間 s-kernel-time 現在の間隔内にサイトが使用した Kernel Mode プロセッサ時間の累計 (%) です。
ページ違反の総数 s-page-faults メモリ ページ違反になったメモリ参照の総数です。
プロセスの総数 s-total-procs 現在の間隔内に作成された CGI およびアウト プロセス アプリケーションの総数です。
有効なプロセス s-active-procs ログが記録されたときに実行されていた CGI およびアウト プロセス アプリケーションの総数です。
切断されたプロセスの総数 s-stopped-procs 現在の間隔内にプロセス調整のために停止された CGI およびアウト プロセス アプリケーションの総数です。

値 意味
Site-Stop Web サイトが何らかの理由で停止されました。
Site-Start Web サイトが開始または再開されました。
Site-Pause Web サイトが一時停止されました。
Periodic-Log 定期的に定義されるログ エントリです。収集の間隔は管理者が指定します。
Reset-Interval-Start リセット間隔が開始しました。
Reset-Interval-End リセット間隔が満了し、リセットされました。
Reset-Interval-Change Web サイトの管理者がリセット間隔の値を変更しました。
Log-Change-Int/Start/Stop 次のイベントのいずれかが発生しました。ログ間隔の変更。間隔イベントの実行。サイトの停止、開始、または一時停止。
Eventlog-Limit CGI またはアウト プロセス アプリケーションが管理者の設定したイベント ログの制限値に達したために、Web サイトのイベント ログが作成されました。
Priority-Limit CGI またはアウト プロセス アプリケーションが管理者の設定した低優先度の制限値に達したために、その優先順位が低く設定されました。
Process-Stop-Limit CGI またはアウト プロセス アプリケーションが管理者の設定したプロセス停止の制限値に達したために、Web サイトがアプリケーションを停止しました。
Site-Pause-Limit CGI またはアウト プロセス アプリケーションが管理者の設定したサイト一時停止の制限値に達したために、Web サイトが一時停止されました。
Eventlog-Limit-Reset リセット間隔が満了になりました。または、イベント ログの制限が手動でリセットされました。
Priority-Limit-Reset リセット間隔が満了になりました。または、優先度の制限が手動でリセットされました。
Process-Stop-Limit-Reset リセット間隔が満了になりました。または、プロセス停止の制限が手動でリセットされました。
Site-Pause-Limit-Reset リセット間隔が満了になりました。または、サイト一時停止の制限が手動でリセットされました。

Microsoft IIS ログ定義

フィールド 説明
クライアントの IP アドレス 要求を行ったクライアント マシンの IP アドレスです。
ユーザー名 サーバーにアクセスした、認証されているユーザーの名前です。ハイフン (-) で表される匿名ユーザーは含まれません。
日付 アクティビティが発生した日付です。
時間 アクティビティが発生した時刻です。
サービスとインスタンス Web サイト インスタンスは W3SVC#、FTP サイト インスタンスは MSFTPSVC# と表示されます。# は、サイトのインスタンスです。
コンピュータ名 サーバーの NetBios 名です。
サーバーの IP アドレス 要求が処理されたサーバーの IP アドレスです。
所要時間 処理にかかった時間です。
送信バイト数 サーバーに送信されたバイト数です。
受信バイト数 サーバーから受信したバイト数です。
サービスのステータス コード HTTP または FTP のステータス コードです。
Windows のステータス コード Windows 関連の処理の状態です。
要求の種類 サーバーが受信した要求の種類です (例、GET および PASS)。
操作の対象 操作の対象であった URL です。
パラメータ スクリプトに渡されるパラメータです。

NCSA 共通ログ ファイル形式

フィールド 説明
リモート ホスト アドレス 要求を行ったクライアント マシンの IP アドレスです。
リモート ログ名 この値は常にハイフン (-) です。
ユーザー名 認証されたユーザーの場合は Domain\username の形式で、匿名ユーザーの場合はハイフン (-) です。
日付 アクティビティが発生した日付です。
時刻と GMT オフセット アクティビティが発生した時刻と GMT オフセットです。
要求およびバージョン 使用された要求の種類、対象であった URL、スクリプトに渡されたパラメータ (ある場合)、およびクライアントが使用した HTTP バージョンです。
サーバーのステータス コード HTTP のステータス コードです。
送信バイト数 サーバーがクライアントに送信したバイト数です。
