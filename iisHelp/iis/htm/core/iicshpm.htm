IIS Global、Web Total Rejected Async I/O Requestsサービスの開始後に、帯域幅の設定により拒否されたユーザー要求の総数です。ブロックされた要求とは異なり、拒否された要求はバッファには保持されません。
Web Total Search Requestsサービスの開始後からの、MS-SEARCH メソッドを使用した HTTP 要求の総数です。Search 要求は、サーバーへの照会に使用され、クライアントが提示する一連の条件に一致するリソースを検出します。全体を計測する場合と、特定のサイトを計測する場合があります。
Web Total Trace RequestsTRACE メソッドを使用する HTTP 要求の数です。
Web Total Unlock Requestsサービスの開始後からの、UNLOCK メソッドを使用した HTTP 要求の総数です。Unlock 要求は、ファイルのロックを解除するために使用されます。全体を計測する場合と、特定のサイトを計測する場合があります。
IIS Global Total URIs CachedWWW サービスと FTP サービスでこれまでにキャッシュに追加された URI 情報ブロックの総数です。
Web Trace Requests/SecTRACE メソッドを使用する HTTP 要求の 1 秒あたりの数です。
ASP Transactions Aborted中止されたトランザクションの数です。
ASP Transactions Committedコミットされたトランザクションの数です。
ASP Transactions Pending処理中のトランザクションの数です。
ASP Transactions Totalサービス開始後のトランザクションの総数です。
ASP Transactions/Sec開始したトランザクションの 1 秒あたりの数です。
Web Unlock Requests/SecUNLOCK メソッドを使用する HTTP 要求の 1 秒あたりの数です。Unlock 要求は、ファイルのロックを解除するために使用されます。全体を計測する場合と、特定のサイトを計測する場合があります。
IIS Global URI Cache Flushesサーバー開始後にフラッシュされた URI キャッシュの数です。
IIS Global URI Cache HitsURI キャッシュへの参照が成功した数の合計です。
IIS Global URI Cache Hits %キャッシュ要求の全体に対して URI キャッシュがヒットした割合です。
IIS Global URI Cache MissesURI キャッシュへの参照が失敗した数の合計です。
