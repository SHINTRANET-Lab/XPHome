[ISAPI フィルタ] プロパティ シート

このプロパティ シートは、ISAPI フィルタのオプションを設定するときに使用します。ISAPI フィルタは、HTTP 要求を処理している間にイベントに応答するプログラムです。

ISAPI フィルタを追加するには、[追加] をクリックします。[フィルタ] 列に区別しやすいフィルタ名を入力し、[実行可能] 列にフィルタへのパスを入力します。

ISAPI フィルタを削除するには、フィルタを選択して、[削除] をクリックします。

ISAPI フィルタのプロパティを変更するには、フィルタを選択し、表示される [フィルタ] および [実行可能] ボックスを必要に応じて変更します。

ISAPI フィルタの実行順序を変更するには、フィルタを選択して、[上へ] または [下へ] をクリックします。優先度が同じフィルタの間でのみ、ロード順序を変更することができます。

注   マスタ プロパティ レベルで構成される ISAPI フィルタはグローバル、つまりサーバー上に作成されるすべての Web サイトに適用されます。マスタ プロパティ レベルの ISAPI フィルタと、個々の Web サイト レベルのフィルタは異なっている場合があります。また、グローバル フィルタは、Web サイト レベルでは表示されません。
