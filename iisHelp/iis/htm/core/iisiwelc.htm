サーバー側インクルード

"サーバー側インクルード (SSI)" ディレクティブは、別ファイルの内容を HTML ページに挿入するよう、Web サーバーに命令します。インクルード ファイルを使うと、多くの Web ページで使う情報を単一のファイルに格納できます。たとえば、SSI ディレクティブを使って、Web サイトのすべてのドキュメントに著作権情報を挿入できます。

ここでは、次の項目について説明します。

  • 「サーバー側インクルードについて」では、サーバー側インクルード ディレクティブについて説明します。
  • 「サーバー側インクルードを有効にする」では、SSI ディレクティブの処理を有効にする方法について説明します。
  • 「サーバー側インクルード ディレクティブの使い方」では、SSI ディレクティブを HTML ページに挿入する方法について説明します。
  • 「サーバー側インクルード リファレンス」では、SSI ディレクティブを使用するときのリファレンス情報を紹介します。
