[FTP サイト] プロパティ シート

このプロパティ シートは、サーバーの識別パラメータを設定するときに使用します。サーバー名、ドメイン名、および IP アドレスは省略可能です。

[識別]

[説明]

サーバー名には、任意の名前を使用できます。この名前は、主にサーバーを識別し、整理するために使用します。

[IP アドレス]

このボックスにアドレスを表示するには、あらかじめコントロール パネルでアドレスを定義しておく必要があります。詳細については、Windows のマニュアルを参照してください。

[TCP ポート]

サービスを実行するポートを指定します。

既定値はポート 21 です。ポートの値は、任意の重複しない TCP ポート番号に変更できますが、あらかじめクライアントにそのポート番号を通知しておく必要があります。通知せずに変更すると、クライアントはサーバーに接続できなくなります。

[接続]

[無制限]

サーバーへの同時接続数を無制限にします。

[最大接続数]

サーバーへの同時接続を許可する最大数を設定します。

[接続のタイムアウト]

サーバーが、反応のないユーザーの接続を切断するまでの時間を設定します。これにより、HTTP プロトコルが接続の解除に失敗しても、すべての接続が切断されます。

[ログを収集する]

このチェック ボックスをオンにすると、FTP サイトのログ収集が有効になります。これにより、ユーザーの利用状況を詳細に記録し、さまざまな形式でログを作成できます。ログの収集を有効にし、[アクティブ ログ形式] ボックスでログの形式を選択します。選択できる形式は、次のとおりです。

  • [Microsoft IIS ログ ファイル形式] 固定の ASCII 形式です。
  • [W3C 拡張ログ ファイル形式] カスタマイズ可能な ASCII 形式です。既定で選択されています。
  • [ODBC ログ] (Windows 2000 Server でのみ使用可能) データベースにログ収集される固定の形式です。

ログ ファイルの作成条件 (毎週、ファイル サイズ別など) を構成する場合、または W3C 拡張形式や ODBC 形式のログ収集のプロパティを構成する場合は、[プロパティ] をクリックし、設定します。
