[パフォーマンス] プロパティ シート

このプロパティ シートは、メモリおよび帯域幅の使用に関連するプロパティを設定する場合に使用します。

[チューニング]

サイトへの 1 日の予想接続数に合わせて、この設定を調整します。設定した数が実際の接続数をわずかに上回っていれば、接続は高速に処理され、サーバーのパフォーマンスが向上します。設定した数が実際の接続数を大きく上回っている場合は、サーバーのメモリが無駄に使用され、サーバー全体のパフォーマンスが低下します。

[帯域幅調整を使用可能にする]

このチェック ボックスをオンにすると、Web サイトで使用する帯域幅が制限されます。この Web サイトに限り、ここで入力した帯域幅の値がコンピュータ レベルで設定された値よりも優先されます。この値が、コンピュータ レベルでの設定値を超える場合でも同様です。

[プロセス調整を使用可能にする]

このチェック ボックスをオンにすると、この Web サイトのアウト プロセス アプリケーションに対する CPU の使用率が制限されます。このチェック ボックスをオンにして、[制限を設定する] チェック ボックスをオフにすると、割り当てられた使用率を超過している場合のみ、イベント ログにイベントが書き込まれます。

[制限を設定する] チェック ボックスをオンにすると、制限を超過する場合があります。サイトが、割り当てられた使用率を 150% 以上超過すると、アプリケーション プロセスがアイドルに設定されます。200% 以上超過した場合は、Web サイト上のすべてのアウトプロセス アプリケーションが停止します。
