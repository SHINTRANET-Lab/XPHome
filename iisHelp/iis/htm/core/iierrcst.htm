カスタム エラー メッセージ

既定の HTTP 1.1 エラー メッセージの代わりにカスタム エラー メッセージを送信するように、IIS を構成できます。カスタム エラー メッセージは、ファイル名または URL にマップできます。

ここで説明する項目は、次のとおりです。

  • 「カスタム エラー メッセージについて」では、既定の HTTP 1.1 エラーおよびカスタマイズした HTTP 1.1 エラーの概要を説明します。
  • 「カスタム エラー メッセージを追加する」では、カスタム エラー メッセージを Web サイトに追加する方法を説明します。
  • 「詳細なカスタム エラー メッセージを有効にする」では、IIS に含まれる詳細で "わかりやすい" HTTP 1.1 エラーの使用方法を説明します。
  • 「ASP エラー処理を有効にする」では、500-100.asp カスタム エラー ファイルを使用した ASP エラーの処理について説明します。

関連するその他の項目

  • Web サイト コンテンツについては、「Web サイト管理」を参照してください。
  • Windows イベントについては、「IIS の Windows イベント」を参照してください。
