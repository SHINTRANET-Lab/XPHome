Internet Explorer 5.0 による発行

Windows XP、Windows 2000、Windows NT 4.0、Windows 98、または Windows 95 のオペレーティングシステムで Internet Explorer 5 により WebDAV ディレクトリへの接続が可能です。Windows 2000 による接続後の操作と同じように、Internet Explorer 5 による接続後、ファイルを操作しディレクトリに発行します。

Internet Explorer による WebDAV ディレクトリへの接続
  1. Internet Explorer 5 の [ファイル] メニューで、[開く] をクリックします。
  2. [開く] ボックスに、接続する WebDAV ディレクトリの URL を入力します。
  3. [Web フォルダを開く] チェック ボックスをオンにし、[OK] をクリックします。ディレクトリにファイルの一覧が表示されます。
