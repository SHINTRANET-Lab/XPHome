[サービスのプロパティ] プロパティ シート

このプロパティ シートは、HTTP 圧縮を有効にする場合や、IIS 3.0 の管理インターフェイスを使用するアプリケーションからのプログラム管理が可能な Web サイトを 1 つ指定する場合に使用します。

[既定の Web サイト]

IIS 3.0 の管理インターフェイスがアクセスする Web サイトを 1 つ選択します。

[HTTP 圧縮の構成]

圧縮に対応するクライアントへファイルを送信する場合は、事前に HTTP 圧縮を使ってファイルを圧縮します。ファイルが要求されると、そのファイルの圧縮コピーがキャッシュ ディレクトリに格納され、それ以降に要求があった場合は、その圧縮コピーが送信されます。一度も要求のないファイルは圧縮されません。

静的 HTML ファイルを圧縮するには、[静的ファイルの圧縮] チェック ボックスをオンにします。

アプリケーション ファイルを圧縮するには、[静的ファイルの圧縮] と [アプリケーション ファイルの圧縮] の両方のチェック ボックスをオンにします。

圧縮ファイルをキャッシュするディレクトリ名とパスを、[一時フォルダ] ボックスに入力します。[参照] をクリックして、ディレクトリを選択することもできます。このディレクトリは、ローカル ハード ディスク上の NTFS パーティションに置かれていなければなりません。このディレクトリを共有、または圧縮することはできません。

キャッシュ ディレクトリのサイズは、[制限] ボックスにサイズを入力することで制限できます。
