テキスト エディタで Web ページを作成する

Web ページは、メモ帳などの標準のテキスト エディタで、HTML タグ を使用して作成できます。HTML タグは、Web ブラウザにドキュメントの表示方法を指示する書式命令です。HTML のタグは、2 つの不等号 (< と >) で囲まれた語句が対になったもので構成されます。たとえば、<B>Hello World</B> という HTML コードにより、Web ブラウザには "Hello World" という言葉が太字で Hello World のように表示されます。

タグを入れ子にして、複数の属性を使用することもできます。たとえば、斜体を "World" という語句だけに指定し、文全体は太字にするには、<B>Hello <I>World</I></B> と記述します。これで、Hello World のように表示されます。タグを重ね合わせないように注意してください。たとえば、<B>Hello <I>World</B></I> という HTML 構文は誤りであるためブラウザによっては、正常な結果が得られないことがあります。

テキスト エディタで Web ページを作成するには
  1. 通常のテキスト (ASCII) エディタで、重要な HTML ヘッダーと書式情報をブラウザに示す Web ページの基本的な骨格、つまりテンプレートを作成します。次のタグを入力します。
    2. <HTML>
<HEAD>
  <TITLE>  </TITLE>
</HEAD>
  <BODY>

  </BODY>
</HTML>
  2. Web ページのタイトルを <TITLE> タグの間に記述します。このタイトルは、Web ブラウザのタイトル バーに表示され、Web ページには表示されません。タイトルには、Web ページの内容を簡単に説明するものを選びます。
  3. Web ブラウザに表示するテキストやそのほかのコンテンツは、<BODY> タグの間に記述します。
  4. 作成した HTML ファイルに、.htm という拡張子を付けて保存します。テキスト エディタの機能により、ドキュメントに .txt 拡張子が自動的に付かないように注意します。次に、Web ブラウザでファイルを開き、変更内容を確認します。
  5. ほかに必要な要素または書式設定を Web ページに追加します。次の HTML タグの表を参考にしてください。手順 4. を繰り返し、変更内容を確認します。

ヒント   HTML を理解して十分に使いこなすには、インターネット上のさまざまな Web サイトから参照できる HTML 仕様を詳しく調べる必要があります。さらに、Web ブラウザのソース表示機能を使えば、ほとんどの Web ページの HTML ソース ファイルを表示することができます。この機能を利用すれば、ほかの HTML 作成者が Web ページの設計に使用している手法を習得できます。

 

HTML タグ 説明
<P> </P> 段落の開始と終了を指定します。
<Hn> </Hn> 見出しの開始と終了を指定します。値 n は、1 から 6 までの整数で、異なるフォント サイズを指定します。
<U> </U> 下線付きテキストの開始と終了を指定します。
<UL> </UL> 番号なしリストの開始と終了を指定します。リストの各項目は、<LI> タグで始めます。
<OL> </OL> 番号付きリストの開始と終了を指定します。リストの各項目は、<LI> タグで始めます。
<LI> リストの項目を示します。終了タグは必要ありません。
<B> </B> 太字の開始と終了を指定します。
< IMG SRC=image file> Web ページに画像を挿入します。
< BGSOUND SRC=sound file> ページを読み込んだ後、バックグラウンド サウンドが再生されます。 注   この機能は、Microsoft Internet Explorer Version 2.0 以降でのみサポートされます。
<FONT SIZE=n FACE=font type> < /FONT > サイズ、色、書体など、テキストのフォント特性の開始と終了を指定します。
<A HREF=file > </A> ファイル内の別の場所、または別のドキュメントへのハイパーリンクを作成します。たとえば、次の構文で作成されるリンクでは、ユーザーがそのリンクをクリックすると、別の HTML ドキュメントが表示されます。 <a href="mainpage.html"> Go back to main page. </a>
<HTML> </HTML> ファイルの内容が、HTML 言語で記述されていることを示します。普通、HTML ドキュメントの最初と最後のタグとして使用します。
<HEAD> </HEAD> ドキュメント タイトルやそのほかの記述情報などが、HTML ドキュメントのヘッダー情報に含まれています。Web ブラウザおよび Web ページ検索プログラムが、この情報を使用します。このタグの間に文は挿入しません。
<TITLE> </TITLE> HTML ドキュメントのタイトルを示します。タイトルは、Web ブラウザのタイトル バーに表示されますが、本文中には表示されません。
<BODY> </BODY> Web ページに表示する内容が入ります。
