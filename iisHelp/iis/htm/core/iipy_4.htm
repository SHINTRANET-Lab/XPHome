[IP アドレス アクセス制限]

このダイアログ ボックスは、Web サイト、仮想ディレクトリ、またはディレクトリへのアクセスを、特定のユーザーに対して許可または拒否するために使用します。

[既定では、すべてのコンピュータからのアクセスを] オプション

[許可する]

例外として指定したコンピュータを除く、すべてのコンピュータにリソースへのアクセスを許可する場合に、このオプションを選択します。たとえば、特定の IP アドレスからサーバーへのアクセスを拒否することで特定の個人またはグループを除外したり、ネットワーク全体に対してサーバーへのアクセスを拒否する一方で、ほかのすべてのネットワークにアクセスを許可することができます。

[拒否する]

アクセスが許可された特定の IP アドレスを使用するユーザーを除く、すべてのリモート ユーザーからのアクセスを拒否する場合に、このオプションを選択します。インターネットでは、既知のユーザー以外の全員を除外する場合に、IP アドレスによるセキュリティを利用すると便利です。

[ただし次のものは除きます]

特定のコンピュータまたはコンピュータ グループからのアクセスを許可または拒否するには、次の操作を行います。

  1. [追加] をクリックします。[許可する] が選択されている場合は、[拒否] が表示され、[拒否する] が選択されている場合は、[許可] が表示されます。
  2. [IP アドレス] 列に、サイトへのアクセスを許可または拒否する特定のコンピュータの、固有の TCP/IP ネットワーク アドレスを入力します。
  3. [サブネット マスク] 列に、サイトへのアクセスを許可または拒否するコンピュータ グループの TCP/IP ネットワーク区分を示す数値を入力します。
  4. [ドメイン] 列に、サイトへのアクセスを許可または拒否するネットワークのドメイン名を入力します。

注   IP アドレスの制限項目を設定するために、[IP アドレス]、[サブネット マスク]、および [ドメイン] の 3 つのテキスト ボックスが用意されていますが、この 3 つすべてを同時に指定することはできません。

1 台のコンピュータへのアクセスを制限する場合は、IP アドレスのみを指定します。

IP の範囲を指定してコンピュータ グループへのアクセスを制限する場合は、IP アドレスとサブネット マスクのみを指定します。

ドメインを指定してコンピュータ グループへのアクセスを制限する場合は、ドメインのみを指定します。
