FTP メッセージとディレクトリ表示スタイルを設定する

FTP サイトをセットアップするときに、サイトの詳細情報をユーザーに示すメッセージを設定することができます。メッセージには、ユーザーが最初に FTP サイトへログオンしたときに表示する接続時のメッセージや、ログオフするときに表示する切断時のメッセージがあります。また、最大接続数に達したことをユーザーに通知するメッセージを表示することもできます (FTP の最大接続数は、[FTP サイト] プロパティ シートで設定できます)。

FTP の接続時、切断時、または最大接続時の各メッセージを設定するには
  1. IIS スナップインで、メッセージを設定する FTP サイトを選択します。
  2. サイトを右クリックして、[プロパティ] をクリックします。
  3. [メッセージ] プロパティ シートをクリックします。
  4. 接続時、切断時、または最大接続時のメッセージを、対応するテキスト ボックスに入力します。

FTP サイトのディレクトリ表示スタイルを設定することもできます。ディレクトリ表示スタイルは、FTP クライアントが LIST 操作を送信したときに使用される形式を定義します。ディレクトリ表示スタイルは、MS-DOS または UNIX のいずれかです。既定のディレクトリ表示スタイルは、MS-DOSです。

既定の MS-DOS ディレクトリ表示の形式は、2 桁の年、月、日です。FtpDirBrowseShowLongDate メタベース プロパティを有効にすることにより、この設定を 4 桁に変更することができます。

UNIX の表示形式では、FTP サーバーの年とファイルの日付 (年) が違う場合、4 桁の年が表示されますが、FTP サーバーの年とファイルの日付 (年) が同じ場合は、年は表示されません。

FTP サイトのディレクトリ表示スタイルを設定するには
  1. IIS スナップインで、ディレクトリ表示スタイルを設定する FTP サイトを選択します。
  2. サイトを右クリックして、[プロパティ] をクリックします。
  3. [ホーム ディレクトリ] プロパティ シートをクリックします。
  4. [ディレクトリの表示スタイル] の [UNIX] または [MS-DOS] をクリックします。
