ASP アプリケーションを構成する

プロパティの中には、Web サーバーにインストールされた ASP アプリケーションごとに設定できるものがあります。たとえば、アプリケーションによるセッションの使用を有効にしたり、既定のスクリプト言語を設定したりすることができます。アプリケーションのプロパティは、個々のページでそのプロパティを直接無効にしない限り、アプリケーションのすべての ASP ページに適用されます。

ASP アプリケーションを構成するには
  1. IIS スナップインで、Web サイトまたはアプリケーションの開始点ディレクトリを選択します。
  2. ディレクトリのプロパティ シートを開き、[ホーム ディレクトリ] タブ、[仮想ディレクトリ] タブ、または [ディレクトリ] タブをクリックします。
  3. [構成] をクリックし、次に [オプション] タブをクリックします。
  4. アプリケーションのプロパティを設定します。各プロパティについては、[オプション] プロパティ シートを参照してください。
