AppUnload

IIsWebDirectory または IIsWebVirtualDir オブジェクトの AppUnload メソッドは、アウト プロセスで実行されている Web アプリケーションをアンロードします。アプリケーションのリソースはすべて解放され、アプリケーションのプロセスは終了します。インプロセスで実行されているアプリケーションは、ほかのアプリケーションから参照されていない場合にのみ解放されます。

注   サーバー コンポーネント DLL は、現在ほかのアプリケーションに参照されていない場合でも、インプロセス アプリケーションから解放されません。

構文

DirObj.AppUnload

 

パラメータ
DirObj
IIsWebDirectory タイプまたは IIsWebVirtualDir タイプの IIS Admin オブジェクトです。
コード例
<% 
  Dim DirObj 
  Set DirObj = GetObject("IIS://LocalHost/W3SVC/1/ROOT/MyAppDir") 
  'Unload the application at this directory. 
  DirObj.AppUnload 
%>
