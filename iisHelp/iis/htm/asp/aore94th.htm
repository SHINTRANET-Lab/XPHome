IIS Admin オブジェクト リファレンス

ここでは、各 IIS Admin オブジェクトのリファレンス ページを示します。各ページでは、ADSI コンテナ情報、オブジェクトの ADsPath、オブジェクトがアクセスできるメタベース プロパティを掲載し、オブジェクトに固有の追加メソッドがあればそれも紹介しています。

注   各 IIS Admin オブジェクトには、KeyType と呼ばれるプロパティがあります。これは、所定の ADSI オブジェクトが表す IIS Admin オブジェクトのタイプを正確に指定するプロパティです。このキー タイプが使用できなかったり、無効である場合は、IIS は一般的なタイプである IIsObject を指定します。KeyType を使用すると、ADSI オブジェクトを有効なオブジェクトに変更し、ほかの IIS 固有のプロパティおよびメソッドにアクセスすることができます。

IIsCertMapper Windows アカウントへ証明書をマップします。
IIsCompressionSchemes HTTP 1.1 圧縮方式のグローバル設定です。
IIsCompressionScheme 個々の圧縮方式の設定です。
IIsComputer IIS 構成のグローバル設定を作成します。
IIsCustomLogModule カスタム ログ収集情報フィールドのノードのプロパティを設定します。
IIsFilter 特定のフィルタに関する情報を提供します。
IIsFilters フィルタを管理します。
IIsFtpInfo IIsFtpService で設定したプロパティに加えて、FTP サーバーの構成プロパティを作成します。
IIsFtpServer 1 台の FTP サーバーの構成プロパティを作成します。
IIsFtpService すべての FTP サーバーに共通する設定プロパティを作成します。
IIsFtpVirtualDir 個々の FTP 仮想ディレクトリのプロパティを設定します。
IIsIPSecurity IP アドレスとドメイン アドレスによるアクセス権の設定に使用できるカスタム ADSI オブジェクトです。
IIsLogModule 特定のログ収集モジュールに関する情報を含みます。
IIsLogModules インストールされたログ収集モジュールに関する情報を含みます。
IIsMimeMap MIME マッピングを管理します。
IIsMimeType 有効な MIME の種類を列挙した一覧を操作します。
IIsWebDirectory Web ディレクトリごとのプロパティを設定します。
IIsWebFile Web ファイルごとのプロパティを設定します。
IIsWebInfo IIsWebService で設定したプロパティに加えて、Web サーバーの構成プロパティを作成します。
IIsWebServer 1 台の Web サーバー の構成プロパティを作成します。
IIsWebService すべての Web サーバーに共通する設定プロパティを作成します。
IIsWebVirtualDir 個々の Web 仮想ディレクトリのプロパティを設定します。
