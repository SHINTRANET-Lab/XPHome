パフォーマンスとセキュリティをテストする

Web アプリケーションの開発プロジェクトには、注意深い計画と開発が不可欠です。しかし、確かな拡張性を備えた Web アプリケーションを作成するためには、拡張性問題について、厳密で定期的なテストを実施することが重要です。Web アプリケーションの開発中に、予想できなかった何らかの拡張性問題が発生することはよくあります。これらの問題を見つけて解決するための唯一の方法は、綿密で十分なテストを実施することです。

一般に拡張性とパフォーマンスの最も有効なテストは、"実用時の" 設定において実際に受けるストレスと類似した状態にアプリケーションを置く方法をとります。

Microsoft® Web Application Stress Tool は、複数のブラウザが Web アプリケーションによってページを要求する状態のシミュレーションを行うためのツールです。このツールによって、各種インターネット サービスのさまざまな負荷を作り出すことができ、Web サイトのパフォーマンスに関するデータ収集を行おうとするユーザーに、豊富な機能を提供します。

Web Application Stress Tool は、Windows 2000 リソース キットに付属の CD-ROM で提供されています。
