Active Server Pages を理解する

Active Server Pages (ASP) を使用すると、Web サイト用の動的なコンテンツの生成、および強力な Web アプリケーションの作成を簡単に行うことができます。ここでは、Web 設計者と Web 開発者のいずれにも有益な ASP の機能を紹介します。

ここでは、次の項目について説明します。

  • 「Active Server Pages の紹介」では、Active Server Pages の概要について説明します。
  • 「ASP の新機能」では、このリリースで追加された新機能について説明します。
  • 「ASP における重要な変更箇所」では、このリリースで変更された機能および動作について説明します。
  • 「ASP チュートリアル」では、ASP を使用してサーバー側のスクリプトを作成する方法を順を追って説明します。
