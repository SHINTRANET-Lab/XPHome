WAMUserPass

このプロパティは、新しく作成された IIS アウト プロセス アプリケーションの COM+ アプリケーション ID として IIS が使用するアカウントのユーザー名を指定します。このプロパティおよびこれに関連する WAMUserName プロパティの値は、IIS のインストール時に設定され、同時に確立される Microsoft Windows ユーザー アカウントのユーザー名およびパスワードの値と一致します。このプロパティの値は変更しないことをお勧めします。変更する場合は、有効な Windows ユーザー アカウントに変更し、WAMUserName は新しいアカウントに対応するパスワードに変更してください。このプロパティおよび WAMUserName の値を変更すると、既存の IIS アウトプロセス アプリケーションの動作が妨げられる場合があります。コンポーネント サービスを使用して、各アプリケーションのプロパティ シートの [Identity] タブにあるユーザー名とパスワード値を編集して、アプリケーション ID を同期化することができます。これらのプロパティ値は、インプロセス アプリケーションには影響しません。
