GetMapping

IIsCertMapper オブジェクトの GetMapping メソッドは、既存の証明書マッピングから証明書およびマッピング データを取得します。マッピングの検索方法には、証明書による検索、名前による検索、Windows アカウントによる検索、および数値文字列のインデックスによる検索の 4 種類があります。取得したマッピングは、指定したパラメータ変数に返されます。

構文

IIsCertMapper.GetMapping IMethod,vKey,vCert,NtAcct,NtPwd,strName,IEnabled

 

パラメータ
IMethod
マッピングの検索に使用する方法を指定します。有効な検索方法は、1、2、3、または 4 です。検索方法 1 は証明書による検索、2 は名前による検索、3 は Windows アカウントによる検索、および 4 は 1 から始まる数値文字列のインデックス ("1"、"2" など) による検索です。
vKey
IMethod で指定した検索で使用するキーを指定します。検索方法 1 の場合は、vKey で証明書を指定します。検索方法 2 の場合は、vKey で名前を指定します。検索方法 3 の場合は、vKey で Windows のアカウントを指定します。検索方法 4 の場合は、vKey で数値文字列のインデックス ("1"、"2" など) を指定します。
vCert
返された証明書を受け取ります。
NtAcct
返された Windows アカウントを受け取ります。
NtPwd
返された Windows アカウントのパスワードを受け取ります。
strName
返された Windows アカウントのわかりやすい名前を受け取ります。
IEnabled
有効なマッピングの場合は戻り値 TRUE を受け取り、無効なマッピングの場合は FALSE を受け取ります。
コード例
<% 
  Dim CertObj, vCert, NtAcct, NtPwd, strName, IEnabled 
  Set CertObj = GetObject("IIS://..path../IIsCertMapper") 
  'Search by Windows account. 
  CertObj.GetMapping 3, "MYACCT", vCert, NtAcct, NtPwd, strName, IEnabled 
%>
