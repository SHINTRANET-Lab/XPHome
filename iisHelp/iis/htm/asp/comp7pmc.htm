Content Linking コンポーネント

Content Linking コンポーネントは NextLink オブジェクトを作成します。このオブジェクトは URL の一覧を管理することで、Web サイトのページを書籍のページのように扱えるようにします。Content Linking コンポーネントを使用すると、目次、および前後の Web ページへのリンクを自動的に生成し、更新できます。これは、オンライン新聞、フォーラム メッセージの一覧などのアプリケーションに理想的な機能です。

Content Linking コンポーネントは、リンクされた Web ページの一覧が入ったコンテンツ リンク リスト ファイルを参照します。このリスト ファイルは、Web サーバーに格納されています。

ファイル名
nextlink.dll Content Linking コンポーネントです。
コンテンツ リンク リスト ファイル Web ページの一覧が入ったテキスト ファイルです。Web ページは表示順に並んでいます。このファイルは、Web サーバーの仮想パスに置く必要があります。

構文

Set NextLink = Server.CreateObject( "MSWC.NextLink" )

 

パラメータ
NextLink
Server.CreateObject を呼び出して作成するオブジェクトの名前を指定します。
メソッド
GetListCount コンテンツ リンク リスト ファイルにリンクされている項目数を計算します。
GetListIndex コンテンツ リンク リスト ファイルに設定されている現在のページのインデックスを取得します。
GetNextDescription コンテンツ リンク リスト ファイルに設定されている次のページの説明を取得します。
GetNextURL コンテンツ リンク リスト ファイルに設定されている次のページの URL を取得します。
GetNthDescription コンテンツ リンク リスト ファイルに設定されている N 番目のページの説明を取得します。
GetNthURL コンテンツ リンク リスト ファイルに設定されている N 番目のページの URL を取得します。
GetPreviousDescription コンテンツ リンク リスト ファイルに設定されている直前のページの説明行を取得します。
GetPreviousURL コンテンツ リンク リスト ファイルに設定されている直前のページの URL を取得します。

例

次の例は、目次を作成します。この例を実行するには、Default Web Site フォルダの Data サブフォルダ内に Content Schedule File を作成する必要があります。たとえば、Default Web Site が C:\Inetpub\Wwwroot にマップされている場合は、C:\Inetpub\Wwwroot\Data\NextLink.txt 内にリスト ファイルを作成する必要があります。

--- NextLink.asp ---

<%
   Set NextLink = Server.CreateObject("MSWC.NextLink")   
   ListFile = "/data/nextlink.txt"
   count = NextLink.GetListCount(ListFile) 
   I = 1 
%> 
<UL>
<% Do While (I <= count) %>  
     <LI>
     <A HREF="<%=NextLink.GetNthURL(ListFile, I)%>"> 
     <%= NextLink.GetNthDescription(ListFile, I) %>
     </A>
     </LI>
     <% I = (I + 1) %>
<% Loop %>  
</UL>

次のスクリプトは、ASP ファイルに Previous Page リンクと Next Page リンクを追加します。 このファイルを、次の行を使用してコンテンツ リンク リスト ファイルに一覧表示されている各ファイルに含めてください。

--- NextLink.inc ---

<BR>
<%
   Set NextLink = Server.CreateObject ("MSWC.NextLink")
   ListFile = "/data/nextlink.txt"
   Index = NextLink.GetListIndex(ListFile)
%>

<TABLE BORDER=0 CELLPADDING=6>
<TR><TD>

<% If (Index > 1) Then %> 
     <A HREF="<%=NextLink.GetPreviousURL(ListFile)%>">
     Previous Page
     </A>
<% Else %>
     Previous Page
<% End If %> 

</TD><TD>

<% If 0 = Index Then %>
     <A HREF="<%=NextLink.GetNthURL(ListFile, 1)%>">
     Next Page
     </A>
<% ElseIf (Index < NextLink.GetListCount(ListFile)) Then %> 
     <A HREF="<%=NextLink.GetNextURL(ListFile)%>">
     Next Page
     </A> 
<% Else %>
     Next Page
<% End If %>

</TD><TD>

<% If Not 0 = Index Then %>
     <A HREF="nextlink.asp">
     Back to Table of Contents
     </A>
<% End If %>

</TD></TR>
</TABLE>
