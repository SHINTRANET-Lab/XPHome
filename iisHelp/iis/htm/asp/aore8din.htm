IIsWebInfo

Web サービスに関連するいくつかのメタベース プロパティは、W3SVC キーの Info サブキーに格納されます。IIsWebInfo オブジェクトは、これらのプロパティの値を設定します。

IIsWebInfo オブジェクトは ADSI オブジェクトですが、ADSI コンテナ オブジェクトではありません。

ADsPath
IIS://MachineName/W3SVC/INFO

MachineName には、任意の名前、または "LocalHost" を指定できます。

構文

varReturn = object.Method

 

要素
varReturn
メソッドからの戻り値を受け取る変数です。
object
通常は、直前の GetObject 操作の結果として、IIsWebInfo オブジェクトを含んでいる変数です。
Method
選択したオブジェクト メソッドです。
プロパティ
ADSI オブジェクト プロパティ  

メタベース プロパティ

AdminServer ServerConfigFlags
CustomErrorDescriptions ServerConfigSSL128
LogModuleList ServerConfigSSL40
ServerConfigAutoPWSync ServerConfigSSLAllowEncrypt

メソッド
ADSI オブジェクト メソッド ADSI オブジェクトの標準メソッドです。
