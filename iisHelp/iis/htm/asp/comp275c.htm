ASP で利用可能なコンポーネント

Active Server Pages (ASP) に含まれるサーバー コンポーネントをスクリプト内で使用することで、動的で対話的な Web ページを作成できます。ここでは、これらのコンポーネントに関する説明とその例を示します。

Ad Rotator コンポーネント AdRotator オブジェクトを作成します。このオブジェクトは、指定されたスケジュールに従って、ページに表示する広告を自動的に交換します。
Browser Capabilities コンポーネント BrowserType オブジェクトを作成します。このオブジェクトは、Web サイトにアクセスする各ブラウザの機能、種類、およびバージョンを判別します。
Content Linking コンポーネント NextLink オブジェクトを作成します。このオブジェクトは、Web ページの目次を作成し、書籍のページのように Web ページを連続的にリンクします。
Content Rotator コンポーネント Web ページ上の HTML コンテンツ文字列を自動的に交換します。
Counters コンポーネント Counters オブジェクトを作成します。このオブジェクトは、個々のカウンタの作成、格納、インクリメント、および取得を実行できます。カウンタの数に制限はありません。
Database Access コンポーネント ActiveX® Data Objects (ADO) を使用したデータベースへのアクセスを提供します。
File Access コンポーネント ファイル入出力へのアクセスを提供します。
IISLog コンポーネント IIS が生成する HTTP アクティビティ ログ ファイルを読み取ることができます。
Page Counter コンポーネント Web ページが開かれた回数を数えて表示します。
Permission Checker コンポーネント Microsoft® インターネット インフォメーション サービス (IIS) が提供するパスワード認証プロトコルを使用して、Web ユーザーにファイルを読み取る権限があるかどうかを判別します。
Status コンポーネント Status オブジェクトを作成します。このオブジェクトは、サーバー ステータス情報を含んだプロパティを持ちます。
Tools コンポーネント Tools オブジェクトを作成します。このオブジェクトは、高度な機能を簡単に Web ページに追加するためのユーティリティを提供します。

ASP を使用したサーバー スクリプト作成の詳細については、「Web アプリケーションを開発する」を参照してください。ASP で提供されている組み込みオブジェクトの詳細については、「ASP 組み込みオブジェクト」を参照してください。

注

このリファレンスの例では、主要スクリプト言語として Microsoft® Visual Basic® Scripting Edition (VBScript) を使用しています。ただし、ASP スクリプトは、サポートされている任意のスクリプト言語で作成できます。主要スクリプト言語を変更する方法については、「スクリプト言語を使用する」を参照してください。
