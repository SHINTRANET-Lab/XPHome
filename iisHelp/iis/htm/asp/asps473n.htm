インストール可能なコンポーネントを使用する

IIS インストール時に、多数のコンポーネントが共にインストールされます。アプリケーションでは、これらのコンポーネントを呼び出してさまざまな作業を実行し、Web サイトを強化することができます。これらのコンポーネントの実際の利点は、最新の Web ブラウザでも結果を表示できることにあります。これは、これらのコンポーネントがサーバー上で実行され、通常の HTML のみをクライアント ブラウザに送信するためです。次の例では、これらのコンポーネントにアクセスする方法と、コンポーネントを利用できる状況をいくつか示します。

次の一覧から例を選択してください。

  • 「Ad Rotator」では、ASP スクリプトで Ad Rotator コンポーネントを使用する方法を示します。
  • 「Browser Capabilities」では、ASP スクリプトで Browser Capabilities コンポーネントを使用する方法を示します。

インストール可能な Active Server コンポーネントの使用に関する詳細、および各コンポーネントの詳細については、「ASP で利用可能なコンポーネント」を参照してください。
