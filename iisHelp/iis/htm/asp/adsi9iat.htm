Delete

ADSI コンテナ オブジェクトの Delete メソッドは、コンテナからオブジェクトを削除します。

構文

Object.Delete (KeyType, Name)

 

要素
object
通常は、直前の GetObject 操作の結果として、IIS Admin オブジェクトを含んでいます。
KeyType
削除する IIS Admin オブジェクトの種類です。
Name
削除するオブジェクトの名前です。
解説

削除するコンテナ内のオブジェクトがアプリケーションの一部である場合、Delete メソッドは、アプリケーションの定義を削除してからオブジェクトを削除します。IIsWebVirtualDir オブジェクトおよび IIsWebDirectory オブジェクトの AppDelete メソッドを参照してください。オブジェクトを削除した後に SetInfo を呼び出す必要はありません。

コード例
<% 
  Dim WebServiceObj 
'Get the Web service object, which contains servers. 
  Set WebServiceObj = GetObject("IIS://LocalHost/W3SVC") 
'Delete the fourth server. 
  WebServiceObj.Delete "IIsWebServer", "4" 
%>
