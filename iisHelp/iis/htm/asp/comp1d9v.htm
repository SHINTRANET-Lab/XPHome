FileExists

FileExists メソッドは、ファイルの存在を確認します。指定された URL が、公開されているディレクトリ内に存在する場合は、–1 を返します。ファイルが存在しない場合は、0 を返します。FileExists は、自分のサイトで公開されているファイルの存在を確認するだけです。このため、絶対 URL ではなく相対 URL を指定します。

構文

oTools.FileExists(URL)

パラメータ
URL
確認するファイルの相対 URL を指定する文字列です。
例

次の例は、FileExists メソッドを使用して、特定のファイルが存在する場合にリンクを作成します。

--- Tools_FileExists.asp ---

<%
Set oTools = Server.CreateObject("MSWC.Tools")

If oTools.FileExists("/win2000.gif") then
%>
  <p>
  <a href="http://www.microsoft.com/windows2000/guide/professional/overview/">
  <img src="/win2000.gif">
  </a>
<%
End If
%>
