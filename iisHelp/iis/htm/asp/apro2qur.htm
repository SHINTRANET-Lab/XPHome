AspLogErrorRequests

このプロパティは、Web サーバーが、失敗したクライアント要求を、Windows イベント ログ ファイルに書き込むかどうかを制御します。AspLogErrorRequests が TRUE に設定されている場合、ASP エラー要求の標準セットがログに記録されます。これらのエラーの一覧については、「ASP エラー一覧」を参照してください。

AspLogErrorRequests が TRUE で、AspErrorsToNTLog が FALSE の場合、ASP エラーは IIS ログに送られます。これらのエラーが深刻である場合、または IIS へのログ記録が前に失敗している場合は、それぞれのエラーは Windows イベント ログに送られます。

AspLogErrorRequests が FALSE の場合、これらのエラーは IIS ログにだけ送られ、Windows イベント ログには送られません。

AspLogErrorRequests および AspErrorsToNTLog が TRUE の場合、ASP 関連のエラーはすべて、Windows イベント ログと IIS ログの両方に送られます。

注   IIS ログ ファイルに ASP エラーをまったく記録しないようにするには、ログ収集すべてを無効にする必要があります。手順については、「DontLog」プロパティおよび「LogType」プロパティを参照してください。

データ型 ブール型
既定値 TRUE
継承 継承可能
