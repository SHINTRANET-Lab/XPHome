IIsCompressionSchemes

IIsCompressionSchemes は、ADSI オブジェクト IIsCompressionScheme で表される、IIS で使用可能な個々の HTTP 1.1 圧縮方式を格納する ADSI コンテナ オブジェクトです。IIsCompressionSchemes には、IIS インストール環境のグローバルな圧縮設定もすべて格納されます。

ADsPath
IIS://MachineName/W3SVC/Filters/Compression/Parameters

MachineName には、任意の名前、または "LocalHost" を指定できます。

構文

varReturn = object.Method

varReturn = object.Property

要素
varReturn
メソッドからの戻り値を受け取る変数です。
object
通常は、直前の GetObject 操作の結果として、IIsCompressionSchemes オブジェクトを参照している変数です。
Method または Property
使用する特定のオブジェクト メソッドまたはプロパティです。
