Hits

Hits メソッドは、指定された Web ページが開かれた回数を返します。

構文

oVar.Hits( [pathInfo] )

 

パラメータ
pathInfo
virtualdirectoryname/filename.asp という形式を使用して Web ページの PATH_INFO を指定する、省略可能なパラメータです。このパラメータを指定しない場合は、現在のページのヒット数が表示されます。
戻り値

指定された Web ページが開かれた回数を示す長整数型の値を返します。

例

次の例は、Hits メソッドを使用して、現在の Web ページが開かれた回数を表示します。カウンタの値をインクリメントするのに PageHit メソッドを使用しない場合、この値は 0 になります。

--- PageCounter_Hits.asp ---

<%
  Set MyPageCounter = Server.CreateObject("MSWC.PageCounter")

  Reset = Request.QueryString("Reset")
  If Not "" = Reset Then 
    MyPageCounter.Reset
    ' or MyPageCounter.Reset([Path])
    Response.Write "<H4>Reset Counter for " & Request.ServerVariables("PATH_INFO") & "</H4>"
  End If

  Hit = Request.QueryString("Hit")
  If Not "" = Hit Then 
    MyPageCounter.PageHit
    ' or MyPageCounter.Reset([Path])
    Response.Write "<H4>Incremented Counter for " & Request.ServerVariables("PATH_INFO") & "</H4>"
  End If
%>

<P>
MyPageCounter.Hits = <%= MyPageCounter.Hits %>
