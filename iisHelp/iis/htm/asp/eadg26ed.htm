IIS アーキテクチャ

IIS は、Windows DNA アーキテクチャに必要不可欠な部分です。IIS の重要な役割は、HTTP を通じてシステムにアクセスしているクライアントを、DHTML、ASP などそのほかの Windows DNA サービスにリンクすることです。さらに、IIS には、カスタム アプリケーション アーキテクチャを定義するために、システム開発者が拡張できる基本的な機能が用意されています。

ここでは、次の項目について説明します。

  • 「IIS のコア機能」では、Web アプリケーションの構築に使用できる基本的な機能について説明します。
  • 「IIS とコンポーネント サービス」では、トランザクション用 ASP アプリケーションの処理の分離、管理、および調整を行うために使用する機能の概要を説明します。
  • 「IIS 要求処理」では、要求の種類を判断するために使用する評価手順について説明します。

ここでは、基本的な機能について説明し、IIS アーキテクチャと、Windows DNA の残りの部分との関係を定義します。また、IIS が HTTP 要求を処理する方法の概要も説明します。
