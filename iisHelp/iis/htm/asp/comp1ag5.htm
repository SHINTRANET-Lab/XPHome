TargetFrame

TargetFrame プロパティは、リンクを読み込むターゲット フレームを指定します。このプロパティは、HTML アンカ ステートメントの TARGET パラメータと同じ機能を果たします。

構文

AdRotator.TargetFrame = frame

パラメータ
frame
広告を表示するフレームの名前を指定します。このパラメータには、_TOP、_NEW、_CHILD、_SELF、_PARENT、_BLANK などの HTML フレームキーワード のいずれかを指定することもできます。既定値は NO FRAME です。
例

次の例は、フレーム AdFrame 内に広告を表示します。

--- AdRot_TargetFrame.asp ---

<%  
 Set ad = Server.CreateObject("MSWC.AdRotator") 
 ad.TargetFrame = "target=_TOP"
%>
<%= ad.GetAdvertisement("/ads/adrot.txt") %>
