KeyType

KeyType プロパティは、メタベース キーのタイプと、このキーに関連する IIS Admin オブジェクトを指定します。このプロパティは IIsWebServer や IIsWebVirtualDir など、キーに関連するオブジェクトのクラスの指定に使用します。KeyType により、このキーでアクセス可能なプロパティが決定されます。

データ型 文字列型
既定値  
継承 継承不可

解説

KeyType プロパティは、すべての IIS Admin オブジェクトからアクセス可能です。このプロパティの値はアクセスした IIS Admin オブジェクトの名前で、ADSI Class プロパティの値と同じです。
