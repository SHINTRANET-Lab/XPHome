ASP 組み込みオブジェクト

Active Server Pages (ASP) は、クラスを実装し、コンポーネントが ASP 組み込みオブジェクトのプロパティおよびメソッドにアクセスできるようにします。ObjectContext オブジェクトは、ASP 組み込みオブジェクトのいずれかに対するインターフェイスを返すメソッドを公開します。コンポーネントはこれらのインターフェイスを使用して、組み込みオブジェクトのメソッドおよびプロパティにアクセスすることができます。

次の表は、組み込みオブジェクト クラスの一覧です。

Class 機能
Application オブジェクト Application オブジェクトのメソッドおよびプロパティを呼び出します。
ASPError オブジェクト ASPError オブジェクトのメソッドおよびプロパティを呼び出します。
ObjectContext オブジェクト 組み込みオブジェクトを返し、トランザクション処理に使用するメソッドを提供します。
Request オブジェクト Request オブジェクトのメソッドおよびプロパティを呼び出します。
Response オブジェクト Response オブジェクトのメソッドおよびプロパティを呼び出します。
ScriptingContext 組み込みオブジェクト、Application、Request、Response、Server、または Session を返します。これは、旧式の方法ですので、代わりに ObjectContext を使用してください。
Server オブジェクト Server オブジェクトのメソッドおよびプロパティを呼び出します。
Session オブジェクト Session オブジェクトのメソッドおよびプロパティを呼び出します。

ObjectContext および組み込みオブジェクトを Visual Basic コンポーネントで使用するには、Visual Basic プロジェクトに Microsoft® ASP Object Library への参照を含める必要があります。
