Border

Border プロパティは、広告の周りを枠で囲むかどうかを指定します。

構文

AdRotator.Border = size

 

パラメータ
size
表示された広告を囲む枠の太さを指定します。既定値は、ロテータ スケジュール ファイルのヘッダーに設定された値です。
例

次の例は、広告を枠なしで表示します。

--- AdRot_Border.asp ---

<%  
  Set ad = Server.CreateObject("MSWC.AdRotator") 
  ad.Border = 0
%>
<%= ad.GetAdvertisement("adrot.txt") %>
