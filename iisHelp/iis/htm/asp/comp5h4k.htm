File Access コンポーネント

File Access コンポーネントは、コンピュータのファイル システムへのアクセスに使用できるメソッドとプロパティを公開します。File Access コンポーネントを使用するには、次のようになるように FileSystemObject オブジェクトを作成します。

<% Set fso = Server.CreateObject("Scripting.FileSystemObject") %>

FileSystemObject の詳細については、Windows Script Technologies を参照してください。[VBScript] または [JScript] をクリックし、[ドキュメント]、[ランゲージ リファレンス] をクリックします。次に右側のフレームの [オブジェクト]、[FileSystemObject オブジェクト] の順にクリックします。
