ADSI リファレンス

ここでは、IIS Admin オブジェクトが公開しているプロパティとメソッドについて説明します。ここでは、次の項目について説明します。

  • 「ADSI オブジェクト プロパティ」では、IIS Admin オブジェクトが実装しなければならない 6 つの基本プロパティ (Name、ADsPath、Class、GUID、Parent、および Schema) について説明します。
  • 「ADSI オブジェクト メソッド」では、IIS Admin オブジェクトが公開しているメソッドの一覧を示します。
  • 「ADSI コンテナ オブジェクト プロパティ」では、IIS Admin コンテナ オブジェクトが追加で実装する属性について説明します。
  • 「ADSI コンテナ オブジェクト メソッド」では、ADSI コンテナ オブジェクトが実装する、IIS Admin オブジェクトとメタベース キーを操作するためのメソッドについて説明します。
  • 「IIS 5.1 の ADSI の変更」は、IIS 5.1 における IIS ADSI プロバイダの変更に関するクイック リファレンスです。
