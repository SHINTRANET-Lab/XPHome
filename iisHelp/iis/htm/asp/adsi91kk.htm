GetObject

ADSI コンテナ オブジェクトの GetObject メソッドは、コンテナ内のオブジェクトにアクセスします。

構文

Set ChildObj = Object.GetObject( Class,ChildName )

 

要素
ChildObj
ASP の GetObject 関数と同じ方法でオブジェクトにアクセスします。
Object
通常は、直前の GetObject 操作の結果として、IIS Admin オブジェクトを含んでいます。
Class
アクセスするオブジェクトのクラスを指定します。
ChildName
アクセスするオブジェクトの名前です。
解説

ChildObj 変数には、コンテナに残っているオブジェクトへのポインタが格納されます。

コード例
<% 
  Dim WebServiceObj, ServerObj 
'Get the Web service object, which contains servers. 
  Set WebServiceObj = GetObject("IIS://LocalHost/W3SVC") 
'Access the object for the third Web server. 
  Set ServerObj = WebServiceObj.GetObject("IIsWebServer", "3") 
%>
