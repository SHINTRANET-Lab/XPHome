@LANGUAGE

@LANGUAGE ディレクティブを使用して、スクリプトのコマンドの解釈に使用する言語を設定できます。スクリプト言語は、IIS にインストールされている任意のスクリプト エンジンに設定できます。既定のスクリプト言語は、VBScript です。したがって、スクリプトに @LANGUAGE ディレクティブを指定していない場合、スクリプトは VBScript エンジンによって解釈されます。

構文

<% @LANGUAGE = scriptengine %>

 

パラメータ
scriptengine
スクリプトをコンパイルするスクリプト エンジンです。IIS には、VBScript と JScript の 2 つのスクリプト エンジンが提供されています。
解説

既定のスクリプト言語は、IIS Admin オブジェクトの AspScriptLanguage プロパティによって変更できます。このプロパティは、Web サービス、Web サーバー、仮想ディレクトリ、または Web ディレクトリに適用できます。詳細については、「IIS Admin オブジェクトを使用する」を参照してください。
