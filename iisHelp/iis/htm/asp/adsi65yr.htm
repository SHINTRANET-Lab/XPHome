ADSI コンテナ オブジェクト プロパティ

ほかのオブジェクトを格納できる IIS Admin オブジェクトも、ADSI コンテナ メソッドおよびプロパティを実装できます。これには、次の読み取り専用の ADSI コンテナ プロパティが含まれます。

ADSI コンテナ プロパティ 説明
_NewEnum 格納されたオブジェクトの取得に使用できる Enumerator オブジェクトを返します。このプロパティによって、VBScript などのオートメーション言語は、"For Each Object in ContainerObject" 構文を使用して、格納されているオブジェクトを列挙できます。
Count コンテナ オブジェクト内のオブジェクト数です。
