管理プロパティ リファレンス

管理プロパティは、メタベースのキーに割り当てられた値と関連付けられています。このリファレンスでは、各プロパティについて、データ型、既定値、継承属性、使用上の注意事項などを説明します。また、各プロパティにアクセスできる IIS Admin オブジェクトの一覧も示します。IIS Admin オブジェクトの詳細については、「IIS Admin オブジェクトを使用する」を参照してください。

注   IIS Admin Base オブジェクト レベルで必要なアルゴリズムを実装するプログラマ向けに、IIS Admin Base オブジェクトの情報を付加しています。

ここでは、次の項目について説明します。

  • 「アルファベット順のプロパティ リスト」からは、各プロパティの解説をすばやく表示できます。
  • 「ADSI オブジェクト データ型と Base オブジェクト データ型」は、IIS Admin Base オブジェクト レベルで必要なアルゴリズムを実装するプログラマ向けのクイック リファレンスです。
