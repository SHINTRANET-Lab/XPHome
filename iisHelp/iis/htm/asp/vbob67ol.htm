ContentType

ContentType プロパティは、応答の HTTP コンテンツ タイプを指定します。ContentType を指定しない場合は、既定値の text/HTML になります。

構文

Response.ContentType[= ContentType]

 

パラメータ
ContentType
コンテンツ タイプを説明する文字列です。通常、この文字列は、type/subtype という形式で指定されます。type はコンテンツの一般的なコンテンツ カテゴリ、subtype は具体的なコンテンツ タイプです。サポートされているコンテンツ タイプの一覧については、Web ブラウザのマニュアル、または現在の HTTP 仕様書を参照してください。
例

次の例は、コンテンツ タイプを、チャンネル定義形式 (CDF) に設定します。

<% Response.ContentType = "application/x-cdf" %>

次の例は、ContentType プロパティにその他の一般的な値を設定します。

<% Response.ContentType = "text/HTML" %>
<% Response.ContentType = "image/GIF" %>
<% Response.ContentType = "image/JPEG" %>
<% Response.ContentType = "text/plain" %>
<% Response.ContentType = "image/JPEG" %>
対象

Response オブジェクト
