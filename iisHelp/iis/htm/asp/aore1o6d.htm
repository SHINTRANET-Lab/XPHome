Pause

IIsWebServer オブジェクトの Pause メソッドは、サーバーを一時停止します。

構文

IIsWebServer.Pause

 

コード例
<% 
  Dim ServerObj 
  'Pause the second Web server. 
  Set ServerObj = GetObject("IIS://LocalHost/W3SVC/2")
  ServerObj.Pause 
%>
関連項目

Start、Continue、Stop、Status
