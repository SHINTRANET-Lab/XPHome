トランザクション処理

トランザクション処理は、アプリケーション開発を達成する方法で、すべての処理はトランザクションと呼ばれる独立した作業単位に分割されます。IIS とコンポーネント サービスの統合により、トランザクション処理をサポートする Web アプリケーションの作成が容易になりました。Web アプリケーションを設計する場合には、決定しなければならない基本的な設計のポイントがあります。ここでは、トランザクション ASP の基本となるテクノロジと共に、基本的な設計の問題について概要を説明します。

ここでは、次の項目について説明します。

  • 「トランザクション Web アプリケーションを設計する」では、トランザクション Web アプリケーション開発における設計上の必要事項について説明します。
  • 「トランザクション処理テクノロジ」では、コンポーネント サービス、およびメッセージ キュー サービス (MQS) について説明します。
