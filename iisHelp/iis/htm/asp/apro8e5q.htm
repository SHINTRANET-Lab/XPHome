NetLogonWorkstation

このプロパティは、Windows XP Professional が動作するコンピュータからのドメイン ログオン制限に対して、ドメイン コントローラが使用する条件を示します。NetLogonWorkstation の値が 0 に設定されている場合は、ドメインおよび Windows Professional のクライアント情報に基づいてログオンが制限されます。NetLogonWorkstation の値が 1 に設定されている場合は、クライアントの IP アドレスに基づいてログオンが制限されます。NetLogonWorkstation の値が 2 に設定されている場合は、クライアントの DNS 名に基づいてログオンが制限されます。

データ型 長整数型
既定値 0 (Windows 2000 Professional 以降の制限のみ)
継承 継承可能

IIS Admin Base オブジェクトの追加情報

次の表は、IIS Admin Base オブジェクトを使用するコードで必要な追加情報の一覧です。

メタベース識別子 MD_NET_LOGON_WKS
既定のビットマスク設定 MD_NETLOGON_WKS_NONE
ユーザー タイプ IIS_MD_UT_SERVER

ビットマスク値
定数 値 説明
MD_NETLOGON_WKS_NONE 0x00000000 Windows 2000 以降の Professional の制限のみ。
