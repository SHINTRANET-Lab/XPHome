GetAdvertisement

GetAdvertisement メソッドは、ロテータ スケジュール ファイルから次の広告を取得します。ユーザーがページを開いたり読み込み直したりしてスクリプトが実行されるたびに、GetAdvertisement メソッドは次に予定されている広告を取得します。

構文

AdRotator.GetAdvertisement( rotationSchedulePath )

 

パラメータ
rotationSchedulePath
ロテータ スケジュール ファイルの位置は、仮想ディレクトリに対する相対パスで指定します。たとえば、物理パスが C:\Inetpub\Wwwroot\Ads\Adrot.txt (ここで、Wwwroot は "/" 仮想ディレクトリです) の場合、パス \Ads\Adrot.txt を指定します。
戻り値

現在のページに広告を表示する HTML を返します。

例

次の例は、すべてのプロパティを使用してユーザーが Web ページを表示するたびに、異なる広告を表示します。

--- AdRot.asp ---

<% 
Set ad = Server.CreateObject("MSWC.AdRotator") 
ad.Clickable = True
ad.Border = 1
ad.TargetFrame = "target=_TOP"
Response.Write ad.GetAdvertisement("adrot.txt") 
%>
