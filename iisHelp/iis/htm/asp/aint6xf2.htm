プログラム管理の選択

「IIS メタベースの紹介」で説明したように、メタベースには、IIS が正常に機能するために必要な構成情報のほとんどが含まれています。このメタベースを、同じように IWamAdmin インターフェイスを使用する IIS Admin Base オブジェクトを使ってプログラム管理することができます。開発者は、このオブジェクトを使用するアプリケーションを作成するか、またはアプリケーションの実装がより簡単な IIS Admin オブジェクト を使用するアプリケーションを作成できます。次の図は、メタベース、IIS Admin Base オブジェクト、および IIS ADSI Admin オブジェクトの間の関係を示しています。

メタベースに格納された情報にプログラムからアクセスするための 2 つの方法が提供されています。

  1. IIS Admin オブジェクト  これらのオブジェクトは、IIS インストール環境の各ブロックを構築するキーに対応しています。つまり、"IIsWebServer" という名前の IIS Admin オブジェクトは Web サーバーに適用されるプロパティとメソッドを公開し、"IIsFtpServer" という名前の IIS Admin オブジェクトは FTP サーバーに適用されます。

    IIS Admin オブジェクトは、Visual Basic、VBScript、JScript などのオートメーション対応言語で使用するのが最も適しています。詳細については、「IIS Admin オブジェクトを使用する」を参照してください。

  2. IIS Admin Base オブジェクト  このオブジェクトは、より高度な開発を行う場合に選択します。詳細については、Platform SDK の 「Advanced Programmatic Administration」 を参照してください。
