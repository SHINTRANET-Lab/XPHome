CustomFields

このプロパティは、現在のログ レコードにある HTTP 操作に含まれていた、補足 HTTP ヘッダーを取得します。補足ヘッダーは、キーと値の組み合わせで構成される 2 次元配列の形式で返されます。CustomFields は、IIS によりログ収集された任意のカスタム フィールドまたは特別なフィールドの取得に使用できます。

構文

var = oIISLog.CustomFields

 

パラメータ
var
可変 2 次元配列です。可変 2 次元配列の各組み合わせは、HTTP ヘッダーを記述するキーと値の組み合わせで構成されます。
例

BytesReceived の例を参照してください。例では、次の 2 つの論理を使用します。

      For i = 1 To UBound(oIISLog.CustomFields)
        Response.Write oIISLog.CustomFields(i,1)
        Response.Write " = " 
        Response.Write oIISLog.CustomFields(i,2)
        Response.Write "<BR>"
