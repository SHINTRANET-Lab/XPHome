Reset

Reset メソッドは、指定された Web ページのヒット数を 0 に設定します。

構文

OVar.Reset([pathInfo])

 

パラメータ
pathInfo
virtualdirectoryname/filename.asp という形式を使用して Web ページの PATH_INFO を指定する、省略可能なパラメータです。このパラメータを指定しない場合は、現在のページのヒット数がリセットされます。

Hits の例を参照してください。
