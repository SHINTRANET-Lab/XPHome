PluginExists

PluginExists メソッドは、指定された Macintosh サーバー プラグインが存在するかどうかを確認します。指定された Macintosh サーバー プラグインが現在登録されている場合は -1 を返し、登録されていない場合は 0 を返します。サーバー プラグインは Macintosh 上でのみ使用されます。IIS for Windows95 以降の場合、PluginExists は常に 0 を返します。

構文

oTools.PluginExists(PluginName)

 

パラメータ
PluginName
サーバー プラグインの名前を指定する文字列です。
関連項目

FileExists、ProcessForm、Random
