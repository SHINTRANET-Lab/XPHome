LogFileLocaltimeRollover

このプロパティは、W3C 拡張ログ収集での IIS ログ ファイルのロールオーバー機能を指定します。このプロパティが FALSE に設定されている場合、既定のロールオーバー機能が有効になります。つまり、現地時間ではなく、協定世界時 (UTC) に基づいて新しいログ ファイルが作成されます。このプロパティが TRUE に設定されている場合、UTC ではなく現地時間に基づいて新しいログ ファイルが作成されます (UTC は、これまでグリニッジ標準時または GMT と呼ばれていました)。

注   このプロパティの設定に関係なく、各 W3C 拡張ログ収集のログ レコードのタイム スタンプは、UTC に準拠しています。

データ型 ブール型
既定値 FALSE
継承 継承可能
