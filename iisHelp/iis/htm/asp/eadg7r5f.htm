開発テクノロジ

Windows DNA は動的なテクノロジの集まりで、Web アプリケーションを構築するために使用できます。Windows XP では、このアーキテクチャにいくつかの主要な機能が追加されました。

ここでは、次の項目について説明します。

  • 「コンポーネント サービス」では、コンポーネントと Web 開発で使用されるサービスについて説明します。
  • 「ダイナミック HTML」では、Dynamic Hypertext Markup Language (DHTML) について説明します。
  • 「Windows スクリプト コンポーネント」では、スクリプト言語を使用して、COM コンポーネントを使用する方法について概要を説明します。
  • 「XML」では、Extensible Markup Language (XML) について説明します。
  • 「Active Directory Service Interfaces (ADSI)」では、Active Directory Service Interfaces (ADSI) と、IIS での使用方法について説明します。
