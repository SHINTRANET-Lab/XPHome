ADSI コンテナ オブジェクト メソッド

IIS Admin オブジェクトの ADSI コンテナ オブジェクト メソッドはメタベース内のキーを操作します。コンテナ オブジェクトの中で IIS Admin オブジェクトを作成、削除、および移動することで、キーを作成、削除、および移動できます。コンテナ オブジェクト メソッドを使用して、仮想ディレクトリやサーバーなど格納されたオブジェクトを列挙することもできます。

IIS Admin オブジェクトおよびメタベース キーを操作する ADSI コンテナ オブジェクト メソッドは、次のとおりです。

メソッド 説明
CopyHere オブジェクトをコンテナにコピーします。
Create コンテナに新しいオブジェクトを作成します。
Delete コンテナからオブジェクトを削除します。
GetObject コンテナ内のオブジェクトにアクセスします。
MoveHere オブジェクトをコンテナにコピーし、コピー元のオブジェクトを削除します。

注   IIS Admin オブジェクトの 1 つがコンテナ オブジェクトである場合、そのオブジェクトは、独自の ADSI オブジェクト プロパティおよびメソッドと共に、IIS 固有のメソッドおよび関連付けされたメタベース プロパティを保持します。
