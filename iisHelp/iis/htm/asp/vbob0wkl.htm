URLEncode

URLEncode メソッドは、指定された文字列に対し、エスケープ文字を含む URL エンコード規則を適用します。

構文

Server.URLEncode( string )

 

パラメータ
string
エンコードする文字列を指定します。
例

次のスクリプトを記述したとします。

<%Response.Write(Server.URLEncode("http://www.microsoft.com")) %>

上の例からは、次の出力が得られます。

http%3A%2F%2Fwww%2Emicrosoft%2Ecom
対象

Server オブジェクト

関連項目
