Win32Error

このメタベース プロパティは、Microsoft Win32 エラー ステータス コードを示します。

データ型 長整数型
既定値 0
既定の継承 継承不可

アクセス場所

この識別子は、すべてのメタベース キーで使用できます。

IIS Admin Base オブジェクトの追加情報

次の表は、IIS Admin Base オブジェクトを使用するコードで必要な追加情報の一覧です。

メタベース識別子 MD_WIN32_ERROR
ユーザー タイプ IIS_MD_UT_SERVER

Microsoft Win32 エラー コードの詳細については、Platform SDK の「Win32 Error Codes」、または MSDN Online の Web サイトを参照してください。
