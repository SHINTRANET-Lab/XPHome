プログラム管理リファレンス

ここでは、IIS で使用可能なプログラムによる管理用のオブジェクトおよびインターフェイスに関するリファレンスを示します。

  • 「IIS Admin オブジェクト リファレンス」は、各 IIS Admin オブジェクト のリファレンスです。
  • 「ADSI リファレンス」では、IIS Admin オブジェクトが公開しているプロパティとメソッドについて説明します。
  • 「管理プロパティ リファレンス」では、データ型、既定値、継承属性などの IIS 管理プロパティについて説明し、使用上の注意事項を示します。

詳細については、Platform SDK の 「Advanced Programmatic Administration」 を参照してください。
