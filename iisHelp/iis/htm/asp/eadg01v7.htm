Web アプリケーションを開発する

このガイドでは、Web アプリケーション設計の概要について説明し、コンポーネントや ISAPI 拡張の構築、データ アクセス、Web ベースのトランザクション、およびパフォーマンスに関するより高度なトピックを紹介します。

ここでは、次の項目について説明します。

  • 「Web アプリケーションの概要」では、IIS プログラミング モデルについて紹介します。
  • 「設計に関する決定事項」では、Web アプリケーションを設計する場合に、考慮しなければならない重要な問題点について説明します。
  • 「ASP でデータにアクセスする」では、Microsoft® ActiveX® Data Objects (ADO) を使用して Web ページからデータにアクセスする方法について紹介します。
  • 「トランザクション処理」では、Windows 2000 以降で提供される IIS サービスとコンポーネント サービスが、Web ベースのトランザクションを提供するためにどのように動作するかの概要を説明します。
  • 「IIS アプリケーションのセキュリティ問題」では、セキュリティ保護されたデータへのアクセス時、および提供時に、Web 開発者が直面する問題について紹介します。
  • 「開発テクノロジ」では、XML やダイナミック HTML などの、最新 Web 技術について復習します。
  • 「拡張性のある Web アプリケーションを開発する」では、Web アプリケーション設計に関する決定事項が、サーバーのパフォーマンスに与える影響について説明します。
