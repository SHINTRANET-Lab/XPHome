Redirect

Redirect メソッドは、ブラウザが別の URL への接続を試みるようにします。

構文

Response.Redirect URL

 

パラメータ
URL
ブラウザのリダイレクト先を指定する URL です。
解説

ページ内で明示的に設定されている応答本文のコンテンツはすべて無視されます。ただし、このページで設定されたほかの HTTP ヘッダーはクライアントに送信され、リダイレクト先の URL がリンクとして含まれる自動応答本文が生成されます。Redirect メソッドは次の明示的なヘッダーを送信します。URL は、メソッドに渡された値です。

HTTP 1.0 302 Object Moved
Location URL
例

次の例は、ユーザーを Microsoft の主要 Web サイトにリダイレクトします。

<% Response.Redirect "http://www.microsoft.com" %>
対象

Response オブジェクト
