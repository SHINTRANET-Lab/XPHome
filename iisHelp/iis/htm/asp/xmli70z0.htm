XML

XML (Extensible Markup Language) は、HTML のようにマークアップをタグ形式でドキュメントに適用します。ただし、XML は一般的なマークアップ言語として使用するように設計されている点が HTML とは異なっています。つまり、XML ドキュメントに適用されたマークアップは、HTML での情報の表示および書式設定の情報を伝えるのに使用できるだけでなく、ドキュメントの意味上の構造や組織上の構造を伝えるのにも使用できます。この柔軟性によって、XML は強力になり、アプリケーションの可能性が広がります。

Microsoft による XML への取り組みの詳細については、MSDN Online にある「XML SDK」を参照してください。XML および XML 関連の標準の詳細については、World Wide Web Consortium Web サイトを参照してください。
