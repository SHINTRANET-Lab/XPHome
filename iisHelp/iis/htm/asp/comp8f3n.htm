Content Linking コンポーネントのメソッド

NextLink オブジェクトに含まれるメソッドは、次のとおりです。

GetListCount コンテンツ リンク リスト ファイルにリンクされている項目数を計算します。
GetListIndex コンテンツ リンク リスト ファイルに設定されている現在のページのインデックスを取得します。
GetNextDescription コンテンツ リンク リスト ファイルに設定されている次のページの説明を取得します。
GetNextURL コンテンツ リンク リスト ファイルに設定されている次のページの URL を取得します。
GetNthDescription コンテンツ リンク リスト ファイルに設定されている N 番目のページの説明を取得します。
GetNthURL コンテンツ リンク リスト ファイルに設定されている N 番目のページの URL を取得します。
GetPreviousDescription コンテンツ リンク リスト ファイルに設定されている直前のページの説明行を取得します。
GetPreviousURL コンテンツ リンク リスト ファイルに設定されている直前のページの URL を取得します。
