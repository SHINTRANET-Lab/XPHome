IIS Admin オブジェクトを使用する

ここで説明する IIS Admin オブジェクトは、プログラムによる管理をできるだけ簡単にするために IIS が提供するものです。IIS Admin オブジェクトは、Microsoft Active Directory Service Interfaces (ADSI) に基づいており、オートメーションに対応します。また、Active Server Pages (ASP) における Microsoft Visual Basic Scripting Edition (VBScript) や JScript、Visual Basic、Java、C++ などの、オートメーションをサポートする任意の言語を使って簡単にアクセスおよび操作できます。

ここでは、次の項目について説明します。

  • 「IIS Admin オブジェクトの概要」では、IIS Admin オブジェクトとその使用方法について説明します。
  • 「IIS Admin オブジェクトを使った管理作業」では、IIS Admin オブジェクトを使って、一般的なプログラム管理作業を実行する方法について説明します。

詳細については、「IIS Admin オブジェクト リファレンス」、「管理プロパティ リファレンス」、「IIS Admin オブジェクト リファレンス」、および「管理プロパティ リファレンス」を参照してください。
