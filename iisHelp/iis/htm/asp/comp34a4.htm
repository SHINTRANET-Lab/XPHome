GetNextURL

GetNextURL メソッドは、コンテンツ リンク リスト ファイル内の次の項目の URL を取得します。

構文

NextLink.GetNextURL( listURL )

 

パラメータ
listURL
コンテンツ リンク リスト ファイルの格納場所です。
戻り値

このメソッドは、コンテンツ リンク リスト ファイル内で指定された、次のページの URL を返します。コンテンツ リンク リスト ファイルに現在のページが指定されていない場合、GetNextURL メソッドは一覧の中の最終ページの URL を返します。

例

次の例は、GetNextURL メソッドを使用して、コンテンツ リンク リスト ファイル内の次ページへのリンクを埋め込みます。GetNextURL メソッドを使用する利点は、コンテンツ ページの順序や数を変更した場合、コンテンツ リンク リスト ファイルの一覧を更新するだけで済むことです。その場合、各ページのリンクを更新する必要はありません。詳細な例は、Content Linking コンポーネントの下部に示されています。

--- NextLink_GetNextURL.asp ---

<%
  ListFile = "/Data/nextlink.txt"
  Set objNextLink = Server.CreateObject("MSWC.NextLink")   
  NextPageURL = objNextLink.GetNextURL(ListFile) 
  ThisPageIndex = objNextLink.GetListIndex(ListFile) 
  If 0 = ThisPageIndex Then
    Response.Write "This page is not listed in your Content Linking List file.<BR>"
  Else
    Response.Write "The URL of the next page is<BR>"
    %><A HREF="<%=NextPageURL%>"><%=NextPageURL%></A><%
  End If
%>
