Tools コンポーネント

Tools コンポーネントは、Tools オブジェクトを作成します。Tools オブジェクトは、高度な機能を簡単に Web ページに追加するためのユーティリティを提供します。

ファイル名
tools.dll Tools コンポーネントです。

構文

Set oTools = Server.CreateObject("MSWC.Tools")

 

 パラメータ
oTools
Server.CreateObject を呼び出して作成する Tools オブジェクトの名前を指定します。

注

パーソナル Web サーバー for Windows® 95 または Windows 98 では、Tools オブジェクトは、既定の仮想ディレクトリ内の Global.asa ファイルに既に組み込まれています。Tools.FileExists、Tools.ProcessForm、および Tools.Random を呼び出すことで、Tools オブジェクトを組み込みオブジェクトであるかのように使用することができます。

メソッド
FileExists ファイルの存在を確認します。
Owner 現在のユーザーがサイトの所有者かどうかを確認します。
PluginExists サーバー プラグインの存在を確認します (Macintosh のみ)。
ProcessForm HTML フォームを処理します。
Random 乱数 (整数) を生成します。
