ASP スクリプトの例

Active Server Pages (ASP) は、IIS の主要な機能の 1 つです。ASP によって、プログラミングの経験がほとんどないユーザーでも、有用な対話型の Web ベース アプリケーションをこれまでにないほど簡単に作成できるようになりました。経験を積んだユーザーであれば、ASP によって洗練された強固な Web アプリケーションを短時間で作成できます。その場合、C++ や Java などの従来のプログラミング言語環境での開発に必要な、大量のリソースを確保する必要はありません。

ここでは、サーバー側スクリプト作成の全般的な機能の概要と ASP の詳細について説明します。同時に、ASP で何ができるかを模索する熟練開発者のために、より高度な例を提供します。

各サンプル スクリプトには、次の処理を行うことができるメニュー バーが用意されています。

  • サンプルの処理内容、サンプルの機能、および検索対象を調べるための概要を読み取ります。
  • ソースを表示します。VBScript 版と JScript 版の両方で提供されるサンプル スクリプトは、正確に、そして綿密に構成されています。
  • サンプルの VBScript 版を実行します。

ここで提供するサンプル スクリプトは、いくつかのカテゴリに分類されます。

  • 「簡単なスクリプト」では、スクリプト構文、フロー制御、関数、およびプロシージャの基本について学びます。
  • 「ASP アプリケーションを作成する」では、Web ベース アプリケーションの構造と機能を紹介するサンプルを学習します。
  • 「ASP による Web 対話機能」では、cookie の組み込みやクライアント側スクリプト作成などのテクニックを使用して、より有用で柔軟なアプリケーションを開発する方法を学びます。
  • 「インストール可能なコンポーネントを使用する」では、IIS と共にインストールされる Active Server Pages コンポーネントの機能を利用する方法を学びます。
  • 「データベースの接続」では、付属のデータベース アクセス コンポーネント ActiveX® Data Objects (ADO) の機能を示す例を学習します。
  • 「ASP トランザクション サービス」では、コンポーネント サービスと ASP スクリプトの間の連携を示すサンプルを学習します。

ここで紹介するサンプルは学習用ですが、ASP スクリプトやアプリケーションの作成に関するチュートリアルではありません。詳細については、「Active Server Pages」および「Web アプリケーションを開発する」を参照してください。
