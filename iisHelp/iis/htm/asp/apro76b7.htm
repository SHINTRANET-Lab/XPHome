ADSI オブジェクト データ型と Base オブジェクト データ型

ADSI プロパティ データ型はスクリプト作成レベルで使用できますが、IIS Admin Base オブジェクト データ型は、IIS Admin Base オブジェクトを使用するコードを記述する時にだけ使用します。この違いは、Microsoft® JScript® や、Visual Basic® Scripting Edition (VBScript) などのスクリプト言語を C や C++ のような言語と比較した場合、データ型の実装が異なるところから由来します。

ADSI データ型と対応する IIS Admin Base オブジェクトのデータ型については、次の表を参照してください。

ADSI データ型 Base オブジェクト データ型
長整数型、ブール型、整数型 DWORD_METADATA
文字列型 STRING_METADATA
ExpandSz EXPANDSZ_METADATA
IPSec、NTACL BINARY_METADATA
リスト型 MULTISZ_METADATA
MimeMapList MULTISZ_METADATA
