ADSI コンテナ オブジェクト

ほかのオブジェクトを含むことができる IIS Admin オブジェクトは、IADs インターフェイスに加えて IADsContainer インターフェイスも実装できます。IADsContainer をサポートする IIS Admin オブジェクトの例としては、次のものがあります。

  • IIsWebService および IIsFtpService オブジェクトを含む IISComputer オブジェクト
  • IIsWebDirectory、IIsWebFile、および追加の IIsWebVirtualDir オブジェクトを含む IIsWebVirtualDir オブジェクト

IADsContainer インターフェイスは、オブジェクトがほかのオブジェクトを含むことを可能にするほか、次の作業の実装を可能にします。

  • コンテナ内にオブジェクトを作成する。
  • コンテナからオブジェクトを削除する。
  • コンテナ内のオブジェクト数をカウントする。
  • 含まれているオブジェクトを列挙する。
  • 含まれているオブジェクトにアクセスする。

ほかのオブジェクトを含むことができる IIS Admin オブジェクトは、コンテナ オブジェクトのプロパティとメソッドも実装できます。そのためにこれらのオブジェクトは、IADsContainer インターフェイス、_NewEnum プロパティと Count プロパティ、および GetObject、Create、Delete の各メソッドをサポートします。また、IIS Admin オブジェクトは、IADs インターフェイスの基本プロパティとメソッド、および IIS 固有の、または IIS で拡張されたあらゆるプロパティとメソッドもサポートします。

詳細については、「ADSI リファレンス」を参照してください。
