Clickable

Clickable プロパティは、広告をハイパーリンクとして表示するかどうかを指定します。

構文

AdRotator.Clickable = value

 

パラメータ
Value
広告をハイパーリンクにするかどうかを指定します。パラメータは、次のいずれかの値になります。既定値は TRUE です。
値 説明
TRUE 広告をハイパーリンクとして表示します。
FALSE 広告をハイパーリンクとして表示しません。

例

次の例は、広告をハイパーリンクとしてではなくイメージとしてのみ表示します。

--- AdRot_Clickable.asp ---

<%  
  Set ad = Server.CreateObject("MSWC.AdRotator") 
  ad.Clickable = FALSE
%>
<%= ad.GetAdvertisement("/ads/adrot.txt") %>
