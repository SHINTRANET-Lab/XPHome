Active Server Pages

Microsoft® Active Server Pages (ASP) は、対話型の Web ページの作成、および強力な Web アプリケーションの構築に使用できるサーバー側スクリプトの作成環境です。サーバーは、ASP ファイルの要求を受け取ると、そのファイルに含まれているサーバー側スクリプトを処理し、Web ページを作成してブラウザに送信します。ASP ファイルには、サーバー側スクリプトのほかに、関連するクライアント側スクリプトを含む HTML、およびデータベースへの接続やビジネス ロジックの処理などのさまざまなタスクを実行する COM コンポーネントへの呼び出しも含まれます。

ここでは、Active Server Pages について紹介し、Active Server Pages を使用したスクリプト作成の基礎について説明します。 また、状態を維持する方法など、より複雑なアプリケーションの問題についても検討します。

ここでは、次の項目について説明します。

  • 「Active Server Pages を理解する」では、ASP を紹介し、今回のリリースにおける新機能を説明します。対話型チュートリアルも用意されています。
  • 「ASP ページを構築する」では、スクリプトの記述に必要な基本情報を提供し、基本構文からトランザクション スクリプトの作成までのトピックを紹介します。
