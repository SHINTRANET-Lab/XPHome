CreateMapping

IIsCertMapper オブジェクトの CreateMapping メソッドは、証明書を Windows アカウントにマップします。

構文

IIsCertMapper.CreateMapping vCert, NtAcct, NtPwd, strName, IEnabled

 

パラメータ
vCert
証明書を含んでいます。証明書は、文字列またはバイトの配列のいずれかで、通常は組み込み ASP Request オブジェクト の ClientCertificate コレクションから取得します。
NtAcct
Windows アカウントの文字列を指定します。
NtPwd
NtAcct のパスワード文字列を指定します。
strName
アカウントのわかりやすい名前を指定します。
IEnabled
マッピングを有効にする場合は TRUE、無効にする場合は FALSE を指定します。
コード例
<% 
  Dim CertObj, vCert 
  vCert = Request.ClientCertificate("CERTIFICATE") 
  Set CertObj = GetObject("IIS://..path../IIsCertMapper") 
  CertObj.CreateMapping vCert, "MYACCT", "MYPASS", "My Name", True 
%>
