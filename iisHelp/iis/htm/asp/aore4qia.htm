IIsCertMapper

IIsCertMapper オブジェクトは、クライアント証明書を Windows ユーザー アカウントにマップします。IIsCertMapper は ADSI オブジェクトですが、ADSI コンテナ オブジェクトではありません。ADSI コンテナ オブジェクトの詳細については、「ADSI の機能」を参照してください。

ADsPath
IIS://MachineName/W3SVC/N/IIsCertMapper

MachineName には、任意の名前、または "LocalHost" を指定できます。

構文

varReturn = IIsCertMapper.Method

 

要素
varReturn
メソッドからの戻り値を受け取る変数です。
Method
選択したオブジェクト メソッドです。
メソッド
CreateMapping 証明書を Windows アカウントにマップします。
DeleteMapping 既存の証明書マッピングを削除します。
GetMapping 既存の証明書マッピングを取得します。
SetAcct 既存の証明書マッピングの Windows アカウント文字列に新しい値を設定します。
SetEnabled 既存の証明書マッピングを有効または無効にします。
SetName 既存の証明書マッピングの名前の文字列に新しい値を設定します。
SetPwd 既存の証明書マッピングの Windows パスワード文字列に新しい値を設定します。

関連項目
