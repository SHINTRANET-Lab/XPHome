SetInfo

SetInfo ADSI の SetInfo メソッドは、変更されたプロパティ値をオブジェクトからメタベースに保存します。オブジェクト プロパティの一部の値またはすべての値を変更した後は、SetInfo メソッドで新しい値をメタベースに保存する必要があります。変更した値のみがメタベースに保存され、未変更の値は上書きされません。

構文

object.SetInfo

 

要素
object
通常は、直前の GetObject 操作の結果として、IIS Admin オブジェクトを含んでいます。
コード例
<% 
  Dim IIsObj 
  Set IIsObj = GetObject("IIS://LocalHost/W3SVC/1/Root") 
'Make some changes to properties. 
' . . . 
'Save the changes back to the metabase. 
  IIsObj.SetInfo 
%>

注   AppCreate を呼び出してカスタム オブジェクトを作成しようとするときに、SetInfo の呼び出しに失敗すると、そのカスタム オブジェクトを呼び出せなくなります。これは、パスのみが存在してオブジェクトが存在しないためです。AppCreate または AppCreate2 を呼び出すと準備情報は保持されますが、SetInfo が呼び出されるまでオブジェクトは作成されません。

関連項目
