ダイナミック HTML

Internet Explorer 4.0 から、ダイナミック HTML (DHTML) が導入されました。DHTML を使用すると、より機能豊富な HTML を作成し、クライアントのイベントに応答することができます。DHTML を有効利用するために HTML ページをアップグレードすることにより、ユーザー経験を豊かにするだけではなく、サーバー リソースをより効果的に使用する Web アプリケーションを構築することもできます。

ドキュメント オブジェクト モデル (DOM) のプロパティを設定することにより、DHTML で HTML ページの外観を制御できます。DOM は、Microsoft が World Wide Web Consortium (W3C) に対して標準として提案したモデルです。DHTML は、DOM プロパティを動的に変更するためのイベント モデルを公開しています。次の例は、HTML ページを <DIV> タグでさらに区切り、ユーザーが入力ボタンをクリックしたときに、ウィンドウが分割されるようにディスプレイ スタイル プロパティを設定します。

<SCRIPT LANGUAGE = "VBScript"> 

Sub showit()
  'This subroutine is called when the user clicks a select button.
  'It displays text in the hidden DIV.
  document.all.MyDiv.style.display = ""
End Sub

</SCRIPT>

<DIV ID= "MyDiv" style="display: 'none'" > 
This is some hidden text.
</DIV>
<Select id= "Showit" onclick=showit>

DHTML の詳細については、MSDN Online を参照してください。
