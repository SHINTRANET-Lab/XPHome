ADSI オブジェクト プロパティ

IIS Admin オブジェクトは、ADSI オブジェクトに必要な 6 つの基本プロパティを実装します。これらのプロパティのデータ型は、すべて文字列です。

ADSI プロパティ 説明
Name 基になる名前空間の中で使用されるオブジェクトの名前です。
ADsPath オブジェクトを一意に識別するパスです。GetObject で、オブジェクトを取得するときに使用されます。
Class オブジェクトのスキーマ クラスの名前です。
GUID このスキーマ クラスのオブジェクトの一意な識別子です。IIS は、グローバル一意識別子 (GUID) を使用します。
Parent 親コンテナ オブジェクトの ADsPath です。
Schema スキーマ内でこのスキーマ クラスを表すオブジェクトの ADsPath です。

ADSI オブジェクトのプロパティは読み取り専用で、ADSI の名前空間を管理する ADSI クライアント プログラムが使用します。次の表は、IIsWebService オブジェクトの ADSI プロパティの例です。LocalHost は、IIS を実行するローカル コンピュータの名前が入るプレースホルダです。

ADSI プロパティ 値
Name W3SVC
ADsPath IIS://LocalHost/W3SVC
Class IIsWebService
GUID {8B645280-7BA4-11CF-B03D-00AA006E0975}
Parent IIS://LocalHost
Schema IIS://LocalHost/schema/IIsWebService

ADSI の Name プロパティを使用して、IIS インストール環境の構造を文書化することができます。

IIS Admin オブジェクトがサポートするメタベース プロパティを使用して、IIS インストール環境の構成を制御することができます。「ADSI オブジェクト メソッド」および、個々の IIS Admin オブジェクトがサポートする特定のメタベース プロパティを参照してください。
