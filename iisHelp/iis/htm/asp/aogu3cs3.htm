ADSI オブジェクト

各 IIS Admin オブジェクトは、ADSI 標準によって定義された IADs インターフェイスを実装します。IIS Admin オブジェクトの IADs インターフェイスには、次の基本機能があります。

  • メタベースの特定のノード上の構成プロパティを簡単に設定および取得します。たとえば、特定の Web サイトのアクセス権 (AccessFlags) を変更することができます。
  • オブジェクトの親コンテナ オブジェクトのパスを取得します。
  • ディレクトリ ツリー内でオブジェクト インスタンスを一意に識別する情報をバインドします。
  • オブジェクトのスキーマ定義へのパスを提供します。
  • オブジェクトの名前とタイプを示す識別情報を提供します。
  • 名前空間のプロパティを取得します。
  • 単純で暗黙的なキャッシング システムを提供します。

IIS Admin オブジェクトは、IADs インターフェイスを実装することで、IIS 固有のメタベース プロパティに加えて、Name、ADsPath、Class、GUID、Parent、および Schema の各プロパティをサポートします。また、IIS の構成管理に必要なメソッドに加えて、IADs、GetInfo、SetInfo、Get、Put、GetEx、および PutEx の各メソッドもサポートされます。IIS Admin オブジェクトが提供する ADSI のプロパティとメソッドの詳細については、「ADSI リファレンス」を参照してください。
