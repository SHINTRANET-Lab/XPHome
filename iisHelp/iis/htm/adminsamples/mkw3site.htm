Web サイトの作成 (mkw3site)

概要

Web サイトを作成します。

用法

mkw3site <--RootDirectory|-r ROOT DIRECTORY>
<--Comment|-t SERVER COMMENT>
[--computer|-c COMPUTER1[,COMPUTER2...]]
[--port|-o PORT NUM]
[--IPAddress|-i IP ADDRESS]
[--admin|-a ADMINISTRATOR]
[--HostName|-h HOST NAME]
[--DontStart]
[--help|-?]

パラメータ

IP ADDRESS 新しいサーバーに割り当てる IP アドレスです。(省略可能)
COMPUTER1, COMPUTER2,... サイトを作成するコンピュータを列挙します。このパラメータは、複数のコンピュータを同じ設定で構成する場合に使用します。コンピュータを指定しない場合は、LocalHost を指定したと見なされます。
ROOT DIRECTORY 新しいサーバーのルート ディレクトリへのフル パスです。
SERVER COMMENT Microsoft 管理コンソール (MMC) に表示される名前です。
ADMINISTRATOR サーバーの管理者として割り当てられるユーザーの名前です。
PORT NUM 新しいサーバーのバインド先となるポートの番号です。
HOST NAME このサイトに割り当てるホスト名です。
警告 : サーバーの検索用に DNS が設定されている場合は、ホスト名のみを指定してください。
DONTSTART この引数を指定すると、Web サーバーの作成後にスクリプトが実行されません。

使用例

Mkw3site -rootdirectory d:\someplace -t NewServer -hostname www.domain.com
