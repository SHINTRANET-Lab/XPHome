仮想 Web ディレクトリの作成 (mkwebdir)

概要

Web サーバーに新しい仮想ディレクトリを作成します。

用法

mkwebdir <--computer|-c COMPUTER1,COMPUTER2,...>
<--website|-w WEBSITE>
<--virtualdir|-v NAME1,PATH1,NAME2,PATH2,...>
[--help|-h]

パラメータ

COMPUTER1, COMPUTER2,... 仮想 Web ディレクトリを作成するコンピュータを列挙します。このパラメータは、複数のコンピュータを同じ設定で構成する場合に使用します。コンピュータを指定しない場合は、LocalHost を指定したと見なされます。
WEBSITE ディレクトリを作成する Web サイトです。名前は、所定の優先順位に従って次のいずれかとして指定できます。
サーバーの番号 (1、2、10 など)
サーバーの種類 (My Server)
サーバーのホスト名 (www.domain.com)
IP アドレス (172.16.0.0)
NAME1,PATH1,NAME2,PATH2 作成する仮想ディレクトリの名前とパスです。

使用例

Mkwebdir c LocalHost w "Default Web Site" v CommonImages,D:\Images
