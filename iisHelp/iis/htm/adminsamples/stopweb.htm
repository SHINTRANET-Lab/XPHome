Web サーバーの停止 (stopweb)

概要

Web サーバーを停止します。

用法

stopweb <--ADSPath|-a server1[,server2,server3...]>
[--computer|-c COMPUTER1[,COMPUTER2...]]
[--verbose|-v]
[--help|-?]

パラメータ

-a SERVERx 停止するサーバーの番号です。
-Computer カンマで区切り、コンピュータ名を列挙します。-c Machine1,Machine2,Machine3 のように指定します。指定したすべてのコンピュータに、指定したパスが存在することが前提です。コンピュータを指定しない場合は、LocalHost を指定したと見なされます。

使用例

stopweb -a 1,2

stopweb -c MACHINE1,MACHINE2,MACHINE3 -a 1,2
