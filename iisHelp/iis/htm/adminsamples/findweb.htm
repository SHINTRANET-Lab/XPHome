Web サイトの検索 (findweb)

概要

指定されたコンピュータ上で、指定された Web サイトを検索します。

用法

findweb [--computer|-c COMPUTER] WEBSITE [--help|-?]

パラメータ
Computer 検索するコンピュータです。指定しない場合は、 LocalHost が検索されます。
WEBSITE 検索する Web サイトです。指定しない場合は、"1" が検索されます。
名前は、所定の優先順位に従って次のいずれかとして 指定できます。
サーバーの番号 (1、2、10 など)
サーバーの種類 ("My Server")
サーバーのホスト名 (www.domain.com)
IP アドレス (172.16.0.0)
help このコマンドのヘルプを返します。

使用例

findweb 

findweb -c MACHINE www.domain.com
