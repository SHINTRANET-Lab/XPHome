管理スクリプト

管理スクリプトは、管理者が、コマンドライン スクリプティング ユーティリティ Cscript.exe を使って Web サーバーの設定を変更するために使用するツールです。

ここでは、次の項目について説明します。

  • 「管理スクリプトについて」では、管理スクリプトの機能を説明します。
  • 「スクリプト マニュアル」では、各スクリプトを、その用法やパラメータ情報も含めて説明します。

関連するその他の項目

  • 管理者用のリファレンス情報については、「管理者用リファレンス」を参照してください。
  • ASP スクリプティングについては、「Active Server Pages」を参照してください。
