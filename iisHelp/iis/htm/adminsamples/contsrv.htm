サーバーの再開 (contsrv)

概要

サーバーを再開します。

用法

contsrv <--ADSPath|-a ServerInstance/X[,ServerInstance/Y,ServerInstance/Z...]>
[--computer|-c COMPUTER1[,COMPUTER2...]]
[--verbose|-v]
[--help|-?]

パラメータ

-a ServerInstance/X The instance of the Web server or FTP server.通常は、w3svc/X または msftpsvc/X で、X はサーバー インスタンスの序数です。 IIS の既定のインストールでは、w3svc/1 は、システム ドライブ上の Inetpub\Wwwroot フォルダにマップされる既定の Web サイトを参照します。
-Computer カンマで区切り、コンピュータ名を列挙します。-c Machine1,Machine2,Machine3 のように指定します。
指定したすべてのコンピュータに、指定したパスが存在することが前提です。コンピュータを指定しない場合は、LocalHost を指定したと見なされます。
-verbose 処理中のステータス情報を表示します。
-h このコマンドのヘルプを返します。

使用例

contsrv -a w3svc/1,msftpsvc/2

contsrv -c MACHINE1,MACHINE2,MACHINE3 -a w3svc/1,msftpsvc/2
