インターネット サービス マネージャ (HTML) へのアクセスは Localhost に限られています
セキュリティ対策のため、セットアップではインターネット サービス マネージャ (HTML) へのアクセスを、サーバー コンピュータ (Localhost、IP アドレス 127.0.0.1) のみに許可しています。

対処方法

インターネット サービス マネージャ (HTML) をリモートで使用してサーバーを管理できるようにするには、サーバー上の IIS スナップインを使用して、Web サイト IISADMIN に対する IP アドレスの制限を更新します。

重要    IP アドレスの制限をサーバー上で変更できるのは、IIS が Windows 2000 Server または Windows 2000 Advanced Server プラットフォームで実行されている場合だけです。Windows 2000 Professional プラットフォーム上の IIS を使って、IP アドレスの制限を変更することはできません。

インターネット インフォメーション サービス スナップインで、次の作業を行います。

  1. 接続中の Web サイトを展開します。
  2. IISADMIN 仮想ディレクトリを選択します。
  3. [操作] メニューの [プロパティ] をクリックします。
  4. [ディレクトリ セキュリティ] タブをクリックします。
  5. IP アドレスおよびドメイン名の制限を編集します。
